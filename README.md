# Genetic Prompt Optimization for Code Summarization

Applies a genetic algorithm to evolve optimal LLM prompt templates for code summarization. Prompt components (role, task description, guard parameters, formatting instructions) are treated as genes — crossover and mutation explore the space of combinations, with fitness scored via ROUGE-L against Opus-generated reference summaries.

## Repo Structure
/data
/functions       # 500 self-contained .py files used as model inputs
/references      # paired .txt reference summaries generated by Opus
/src               # GA implementation, fitness function, eval loop
/results           # fitness logs, evolved prompt outputs

## How it works

1. GA evolves prompt templates over the `/data/functions` benchmark
2. Each candidate prompt is scored against `/data/references` using ROUGE-L + length/format penalties
3. GA is compared against a Random Search baseline for statistical validation
