Computes the Levenshtein (edit) distance between two strings `a` and `b`, returning the minimum number of single-character insertions, deletions, or substitutions needed to transform one into the other. It uses the standard dynamic programming recurrence but optimizes memory to O(min-row) by keeping only the previous row instead of a full matrix, running in O(|a|·|b|) time. Early-exit shortcuts handle the cases of equal strings and empty inputs.
