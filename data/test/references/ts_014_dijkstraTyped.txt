This function computes single-source shortest path distances on a weighted directed graph using Dijkstra's algorithm, taking a `Map` adjacency list and a `source` node ID and returning a `Map` from each node to its minimum distance from the source. It initializes all known nodes to infinity (zero for the source), then repeatedly extracts the unvisited node with the smallest tentative distance via a linear scan over a pending set and relaxes its outgoing edges. The implementation runs in O(V² + E) time due to the linear minimum search rather than a priority queue, terminates early when only unreachable nodes remain (leaving their distances as infinity), and assumes non-negative edge weights as required by Dijkstra's correctness.
