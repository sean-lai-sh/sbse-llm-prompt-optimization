The `partitionByPivot` function takes a numeric array `arr` and a `pivot` value, and returns a new array containing the same elements reordered into three contiguous groups: elements less than the pivot, elements equal to the pivot, and elements greater than the pivot. It performs a single O(n) pass using three temporary buckets and concatenates them via spread, implementing the Dutch National Flag-style three-way partition without modifying the input. Relative order within each group is preserved (stable), and edge cases like an empty input or absence of the pivot value naturally yield an empty array or empty middle section, respectively.
