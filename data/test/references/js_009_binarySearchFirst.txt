This function performs a binary search on a sorted array `arr` to find the index of the first (leftmost) occurrence of `target`, returning that index or -1 if not found. When a match is found at `mid`, it records the index but continues searching the left half (`hi = mid - 1`) to locate any earlier occurrence, ensuring the leftmost match is returned in the presence of duplicates. It uses the unsigned right shift `>>>` for the midpoint calculation to avoid potential integer overflow on large indices, and runs in O(log n) time.
