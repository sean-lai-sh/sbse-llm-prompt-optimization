This function implements the classic greedy interval scheduling algorithm to find the maximum number of mutually non-overlapping intervals from an input array of `[start, end]` pairs. It sorts a copy of the intervals by end time in ascending order, then iterates through them, selecting each interval whose start is greater than or equal to the last chosen interval's end (treating touching endpoints as non-conflicting). It returns the selected subset of intervals, or an empty array if the input is empty; the original input is not mutated due to the spread-based copy.
