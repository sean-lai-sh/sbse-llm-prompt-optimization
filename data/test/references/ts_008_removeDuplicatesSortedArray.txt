This function removes duplicates in-place from a sorted numeric array using a two-pointer technique, where `write` tracks the position for the next unique element and `read` scans forward, copying a value only when it differs from the last written one. It mutates the input array by truncating its length to the number of unique elements and returns that count. Edge case: an empty array returns 0 immediately; correctness relies on the input being sorted, since the comparison only checks against the immediately preceding unique element.
