Computes the product of all numeric elements in the input array `arr` by reducing with multiplication, starting from an initial accumulator of 1, and returns the resulting number. Because the seed is 1, an empty array returns 1 (the multiplicative identity), and any element equal to 0 forces the result to 0; non-numeric values will follow JavaScript's standard coercion rules for the `*` operator (e.g., yielding `NaN` for non-coercible values).
