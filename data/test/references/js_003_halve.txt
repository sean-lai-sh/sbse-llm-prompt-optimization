The `halve` function takes a single numeric input `x` and returns its value divided by 2. It performs standard JavaScript floating-point division, so non-integer inputs yield fractional results, and special values propagate per IEEE 754 semantics (e.g., `NaN` returns `NaN`, `Infinity` returns `Infinity`, and non-numeric inputs are coerced via the `/` operator, potentially producing `NaN`).
