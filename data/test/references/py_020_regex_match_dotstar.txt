This function performs full-string regex matching of `text` against `pattern`, returning True iff the pattern matches the entire text. It supports literal characters, `.` (matching any single character), and `*` (zero-or-more of the preceding element), using a bottom-up 2D dynamic programming table of size (m+1)×(n+1) for O(m·n) time and space. The first row is initialized to handle patterns like `a*b*` that can match an empty string by skipping `x*` pairs, and `*` transitions consider both the zero-occurrence case (`dp[i][j-2]`) and the one-or-more case (`dp[i-1][j]` when the preceding pattern char matches the current text char). Note it assumes `*` is never the first character of the pattern (no bounds check beyond `j >= 2` in the initialization row).
