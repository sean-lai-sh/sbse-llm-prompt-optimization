This function decodes a run-length-encoded string where each run is represented as a decimal count immediately followed by a single character (e.g., `"3a2b"` → `"aaabb"`). It iterates through the input, parsing consecutive digits into a count via `Number()`, then appending the next character repeated that many times to the output, returning the fully expanded string. It throws an error if a run lacks a leading digit sequence or if a digit run is not followed by a character (dangling count at end of input).
