This function implements Tarjan's strongly connected components algorithm on a directed graph represented as an adjacency-list object (keys are node identifiers, values are arrays of successor nodes), returning an array of SCCs where each SCC is an array of node keys. It performs a recursive DFS tracking each node's discovery `index` and `lowlink`, using an explicit stack and `onStack` set to identify SCC roots (nodes where `lowlink === index`) and pop their members off the stack as a component. Note that it relies on recursion so deep graphs may overflow the call stack, it iterates only over nodes appearing as keys in `graph` (so successor-only nodes won't be visited unless guarded by the `graph[v] || []` fallback), and node identifiers are coerced to strings via `Object.keys`/`Map` lookups.
