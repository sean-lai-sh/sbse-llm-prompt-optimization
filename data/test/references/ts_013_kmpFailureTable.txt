This function computes the KMP (Knuth-Morris-Pratt) failure table (also known as the longest proper prefix-suffix array) for a given string pattern, returning an array where each entry `table[i]` holds the length of the longest proper prefix of `pattern[0..i]` that is also a suffix. It runs in O(n) time using two pointers: `i` scanning the pattern and `len` tracking the current matched prefix length, falling back via `table[len-1]` on mismatches rather than restarting. Edge cases are handled correctly: an empty pattern returns an empty array, and `table[0]` remains 0 since the loop starts at index 1.
