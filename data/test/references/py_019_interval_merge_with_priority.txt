This function merges a list of (start, end, priority) integer tuples into a minimal set of non-overlapping intervals, returning them sorted by start position. It first sorts the input by (start, end), then sweeps through, extending the current interval's end and taking the max priority whenever the next interval's start is ≤ the current end; otherwise it flushes the current interval and begins a new one. It returns an empty list for empty input, treats touching intervals (s == cur_end) as overlapping, and—since `max` returns the first argument on ties—preserves the earlier-encountered priority when priorities are equal.
