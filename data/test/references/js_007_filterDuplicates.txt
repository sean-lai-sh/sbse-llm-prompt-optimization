This function takes an array `arr` and returns a new array containing only the elements that appear exactly once in the input, preserving their original order. It works in two passes: first building a `Map` of element-to-occurrence-count, then filtering the original array to keep items whose count is 1. Equality is determined by `Map` key semantics (SameValueZero), so `NaN` values are treated as equal to each other, while objects are compared by reference rather than structural equality.
