The `partitionArray` function splits an input array `arr` into two groups based on a `predicate` function, returning a two-element tuple `[truthy, falsy]` where the first array contains items for which the predicate returns a truthy value and the second contains the rest. It iterates through the array once in O(n) time, preserving the original order of elements within each partition. If `arr` is empty, it returns two empty arrays; the predicate's return value is evaluated for truthiness rather than strict equality to `true`.
