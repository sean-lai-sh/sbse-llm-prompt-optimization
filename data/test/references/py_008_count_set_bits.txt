Counts the number of set (1) bits in the binary representation of a non-negative integer `n`, returning the count as an int. It uses Brian Kernighan's algorithm, repeatedly clearing the lowest set bit via `n &= n - 1` until `n` is zero, yielding O(k) time where k is the number of set bits. Raises `ValueError` if `n` is negative; returns 0 for `n == 0`.
