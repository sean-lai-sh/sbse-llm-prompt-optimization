This function iteratively computes the nth Fibonacci number, taking a non-negative integer `n` and returning the corresponding Fibonacci value as a number. It uses a constant-space, O(n)-time loop that maintains two running values (`a` and `b`) representing the previous two terms, updating them in place rather than recursing or allocating an array. It throws an Error if `n` is negative, and short-circuits the base cases by returning `n` directly when `n` is 0 or 1. Note that for sufficiently large `n` the result will exceed JavaScript's safe integer range and lose precision.
