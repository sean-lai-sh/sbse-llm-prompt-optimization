This function groups anagrams from an input array of strings, returning an array of arrays where each inner array contains words that are anagrams of one another. It works by computing a canonical key for each word—the word's characters sorted alphabetically—and using a Map to bucket words sharing the same key, preserving insertion order in both the outer and inner arrays. The algorithm runs in O(n · k log k) time where n is the number of words and k is the maximum word length; edge cases like an empty input array yield an empty result, and empty strings are grouped together under the empty-string key.
