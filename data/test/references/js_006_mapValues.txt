This function returns a new object with the same own enumerable string keys as the input `obj`, where each value is replaced by the result of calling `fn(value, key)`. It iterates via `Object.keys`, so it skips inherited and symbol-keyed properties, and it does not mutate the original object. The output preserves key insertion order from `Object.keys` and produces an empty object when `obj` has no own enumerable string keys.
