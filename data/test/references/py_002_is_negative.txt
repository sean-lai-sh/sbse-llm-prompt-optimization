The `is_negative` function takes a single float argument `n` and returns a boolean indicating whether it is strictly less than zero. Zero itself returns `False`, and negative zero (`-0.0`) also returns `False` since it compares equal to `0.0` in IEEE 754. Note that while typed as `float`, the function will work with any value supporting the `<` operator against `0`, and `NaN` inputs will return `False` due to IEEE 754 comparison semantics.
