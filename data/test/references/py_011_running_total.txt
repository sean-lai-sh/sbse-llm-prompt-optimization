Computes the cumulative sum (prefix sums) of a list of floats, returning a new list of the same length where each element at index i equals the sum of nums[0..i]. The accumulator is initialized to 0.0, so an empty input yields an empty list, and all returned values are floats regardless of input numeric types. The implementation runs in O(n) time and O(n) extra space, leaving the input list unmodified.
