The `parityFlag` function takes a number `n` and returns the string literal `"even"` if `n` is divisible by 2, or `"odd"` otherwise, using the `%` modulo operator. Its return type is the union `"even" | "odd"`. Note that for non-integer inputs it still returns `"odd"` whenever `n % 2` is nonzero (e.g., `1.5` yields `"odd"`), and for negative zero or negative even integers it correctly returns `"even"` since `n % 2` evaluates to `0` or `-0`, both equal to `0` under `===`.
