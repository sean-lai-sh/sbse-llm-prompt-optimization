Recursively merges one or more source objects into the target, mutating target in place and returning it. For each key in a source it descends into nested plain objects on both sides (creating an empty object on the target when needed), and otherwise overwrites the target's value via Object.assign. The function processes sources sequentially via shift() and a tail recursive self-call, so later sources win on conflicts. Arrays are not merged element-wise; they are treated as leaves and replaced wholesale, and the input sources array is mutated by the shift operation.
