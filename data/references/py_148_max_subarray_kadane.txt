Computes the maximum sum of any contiguous subarray of nums using Kadane's algorithm, maintaining a running current best ending at the latest index and a global max_sum across all positions. At each step current is updated to max(n, current + n), discarding the prior subarray when starting fresh yields a larger value, and max_sum tracks the best so far. Runs in O(n) time and O(1) extra space, returns 0 for an empty list, and correctly handles all-negative inputs by returning the largest single element.
