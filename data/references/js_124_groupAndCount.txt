Groups an array of items by a key, returning an object whose keys are the stringified group identifiers and whose values are {items, count} buckets. The key argument can be either a property name or a function applied to each item to derive the group key, which is then coerced via String for use as the object key. Iteration is a single O(n) pass; non-array input returns {}. Because keys are stringified, distinct values that share a string representation (such as 1 and "1") are merged into the same group.
