Reverses the word order of the input string str by trimming leading and trailing whitespace, splitting on one or more whitespace characters via the regex /\s+/, reversing the resulting token array, and joining with single spaces. Runs in O(n) time and O(n) space, returning a new string without mutating the input. Collapses runs of internal whitespace into single spaces and returns an empty string when given a string consisting only of whitespace.
