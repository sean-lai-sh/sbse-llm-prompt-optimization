Performs Knuth-Morris-Pratt substring search, returning a list of every starting index in text where pattern occurs. It first computes the longest-proper-prefix-suffix (lps) array for pattern in O(m) time, then scans text with two pointers i and j that never backtrack, falling back via lps[j-1] on mismatches and recording a match whenever j reaches m. The total runtime is O(n + m) with O(m) extra space, and the function returns an empty list when either the text or pattern is empty.
