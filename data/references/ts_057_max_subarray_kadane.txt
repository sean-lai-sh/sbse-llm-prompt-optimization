Implements Kadane's algorithm to find the contiguous subarray of nums with the maximum sum, returning a SubarrayResult containing maxSum and the inclusive startIndex/endIndex. It maintains a running currentSum that is reset to the current element whenever extending the previous subarray would yield a smaller value, tracking a tentative tempStart so the start index can be committed only when a new maximum is observed. The algorithm runs in O(n) time and O(1) extra space, correctly handles all-negative inputs by seeding from nums[0], and throws a RangeError on an empty array.
