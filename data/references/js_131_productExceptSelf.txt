Computes, for each index i, the product of all elements of nums except nums[i], without using division. It performs two linear passes: a left-to-right sweep that fills result[i] with the running product of elements strictly to the left, then a right-to-left sweep that multiplies in the running product of elements strictly to the right. Time complexity is O(n) and auxiliary space is O(1) excluding the output array. The function does not special-case zeros (the two-pass approach handles them naturally) but assumes nums is a defined array of numeric values.
