Constructs a Binary Indexed (Fenwick) Tree over nums and processes a stream of operations, returning the result of each. It supports prefixSum(i), rangeSum(l, r) computed as prefixSum(r) - prefixSum(l-1), point updates that translate a new value into the appropriate delta and propagate it up the tree, and a findKth helper that uses bit-mask descent to locate the smallest index whose prefix sum reaches k. Initial construction calls update for every element giving O(n log n) build time, while each subsequent query/update is O(log n). The implementation uses 1-indexed internal storage with the standard x & -x lowbit trick and mutates nums on point updates.
