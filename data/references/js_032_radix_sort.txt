Sorts the array of non-negative integers arr in ascending order using LSD radix sort, repeatedly applying a stable counting-sort-by-digit helper for each base-10 digit position from the ones place up to the most significant digit of the maximum value. Returns a new array and short-circuits when the input is empty. Runs in O(d * (n + 10)) time where d is the digit count of the maximum, requires non-negative integer inputs, and would silently misbehave on negatives or floats because the digit extraction uses Math.floor and modulo.
