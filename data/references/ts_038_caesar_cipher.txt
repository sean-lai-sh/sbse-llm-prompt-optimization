Applies a Caesar cipher to a string by shifting each ASCII letter forward by the given amount while preserving case and leaving non-letters untouched. It accepts a text string and a numeric shift and returns the encrypted string in O(n) time and O(n) space. The shift is normalized to the range [0, 25] via ((shift % 26) + 26) % 26 so negative or large positive shifts behave correctly, and only A-Z and a-z are transformed.
