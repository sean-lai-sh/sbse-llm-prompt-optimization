Sorts a shallow copy of arr in ascending order using heap sort, first building a max-heap with a bottom-up sift-down loop and then repeatedly swapping the root with the last unsorted index and re-heapifying the reduced heap. The recursive heapify helper enforces the max-heap invariant at index i within a heap of logical size n. Returns the sorted copy without mutating the original, runs in O(n log n) time in all cases, and uses O(n) extra space for the copy with O(log n) recursion depth.
