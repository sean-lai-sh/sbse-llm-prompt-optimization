Parses a URL query string into a plain object of decoded key/value pairs. It accepts a string (optionally beginning with "?") and returns {} for non-strings or empty queries; otherwise it splits on "&", URL-decodes each side of the first "=", and assigns it into the result. Bare keys without "=" become true, and repeated keys are coalesced into an array, growing to additional elements on subsequent occurrences. Decoding uses decodeURIComponent, so malformed percent-encoding will throw rather than be silently handled.
