Min-max normalizes a number array into the [0, 1] range by computing min and max via Math.min/Math.max applied with spread, then mapping each element to (n - min) / (max - min). Returns an empty array for empty input and an all-zeros array when every element is equal (range of 0) to avoid division by zero. The spread on Math.min/max can overflow the call stack for very large arrays, and the function does not validate against NaN or non-finite inputs.
