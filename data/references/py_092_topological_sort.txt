Computes a topological ordering of a directed graph (adjacency-list dict) using iterative-style DFS with a recursion stack to detect back-edges (cycles). It collects all nodes including those mentioned only as neighbors, visits them in sorted order for deterministic output, appends each node to a result list on post-order, and returns the reversed list as the topological order. When detect_cycle is True and a cycle is found it returns None; when False it breaks early and returns a partial ordering. Runs in O(V + E) time after the O(V log V) sort of node names.
