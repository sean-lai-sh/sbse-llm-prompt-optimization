Reconstructs a nested object from a flat record whose keys are dot-delimited paths. For each path it splits on ".", walks (creating empty objects as needed) along all but the last segment, then assigns the value at the leaf; existing non-object intermediates are overwritten with new objects. The result is always built from plain object nodes, so numeric path segments do not produce arrays, and conflicting paths (e.g. "a" and "a.b") cause the later one to clobber the earlier.
