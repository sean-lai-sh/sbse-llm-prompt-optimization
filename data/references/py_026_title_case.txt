Converts a string to title case using English style guidelines, capitalizing every word except a fixed set of short minor words (articles, conjunctions, and short prepositions like 'a', 'the', 'and', 'of'). It lowercases the input, splits on whitespace, and capitalizes each token unless it is in the minor_words set, with the exception that the first token is always capitalized. Runtime is O(n), the function joins tokens back with single spaces (so original whitespace formatting is normalized), and it does not handle hyphenated words or apostrophes specially.
