Returns a new matrix obtained by rotating the input matrix 90 degrees clockwise, mapping element matrix[r][c] to rotated[c][n-1-r] where n is the original row count. The output is a freshly allocated cols-by-rows grid, so the function works for non-square inputs and does not mutate the original; it runs in O(rows * cols) time and space. An empty matrix or matrix with an empty first row returns an empty list.
