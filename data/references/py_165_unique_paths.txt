Counts the number of distinct paths from the top-left to the bottom-right of an m by n grid moving only right or down, using bottom-up dynamic programming. It initializes a 2D table to all ones (covering the boundary base cases) and fills each interior cell as dp[r][c] = dp[r-1][c] + dp[r][c-1], returning the bottom-right value in O(m * n) time and space. The function does not validate inputs and assumes m and n are at least 1; it would raise an IndexError on dimensions of 0.
