Locates target in the sorted array arr by first checking index 0, then doubling an index i until arr[i] exceeds target or i passes the array end, and finally delegating to a helper binary search bounded by [i/2, min(i, length - 1)]. Returns the index of a match or -1 when the value is absent or arr is empty. Runs in O(log p) time where p is the position of the target, making it especially efficient when the target sits near the start of an unbounded or very large sorted sequence.
