Computes the longest common prefix shared by all strings in an array using horizontal scanning, repeatedly trimming the candidate prefix one character from the right until every string starts with it. It accepts a string[] and returns a string in O(S) time, where S is the total number of characters, and O(1) extra space beyond the prefix. An empty input array yields an empty string, and the inner loop short-circuits to "" as soon as the candidate prefix is fully consumed.
