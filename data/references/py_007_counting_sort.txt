Performs counting sort on a list of integers and returns a newly allocated sorted list. It computes the min and max to size a count array of length max-min+1, tallies each value's frequency offset by min so negative numbers are supported, then expands the counts back into a flat list in order. Runtime is O(n + k) and space is O(k) where k is the value range, the empty input is handled by an early return, and the algorithm only works for integer keys with a bounded range.
