Computes the minimum-cost path sum from the top-left to the bottom-right of a non-negative integer grid, allowing only right and down moves. It builds a 2D DP table the same size as the grid, seeding the first row and column with prefix sums and filling each interior cell with grid[r][c] + min(dp[r-1][c], dp[r][c-1]). The algorithm runs in O(rows * cols) time and space, and returns 0 for an empty grid or grid with empty rows.
