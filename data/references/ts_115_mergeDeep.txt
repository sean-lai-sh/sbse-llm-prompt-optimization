Recursively merges a Partial<T> source into a shallow copy of target, returning a new object without mutating either input. For each source key, it recurses when both the corresponding source and target values are non-null, non-array plain objects; otherwise it overwrites with the source value, treating undefined as a no-op so existing target values are preserved. Arrays are replaced rather than concatenated, and there is no cycle protection, so circular source structures cause infinite recursion.
