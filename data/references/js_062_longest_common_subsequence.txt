Computes the longest common subsequence of two strings text1 and text2 using a standard (m+1) by (n+1) DP table where dp[i][j] is the LCS length of the first i and j characters; matching characters extend the diagonal by one and mismatches take the maximum of dropping a character from either side. After filling the table in O(m*n) time and space, it reconstructs the LCS string by walking back from dp[m][n], stepping diagonally on matches and otherwise moving toward the larger neighbor. The function returns an object with the numeric length and the reconstructed lcs string. Empty inputs are handled correctly via the zero-initialized base row and column.
