Performs wildcard pattern matching where '?' matches any single character and '*' matches any (possibly empty) sequence, using a bottom-up (m+1) by (n+1) DP table with dp[i][j] true when s[0..i-1] matches p[0..j-1]. The base row propagates dp[0][j] = dp[0][j-1] only while leading characters in p are '*', breaking immediately on any literal; the recurrence treats '*' as either an empty match (dp[i][j-1]) or an extension consuming one more character (dp[i-1][j]), and '?' or matching literals as dp[i-1][j-1]. Total time and space are O(m*n). It performs full-string matching, returns false when the pattern lacks a wildcard and lengths differ, and treats unmatched cells as their default false.
