Returns all modal values of the numeric array nums (the values tied for highest frequency) in O(n) time and O(k) space where k is the number of distinct values. It builds a frequency map by iterating once over nums, then computes the maximum frequency via Math.max(...Object.values(freq)) and filters keys whose count equals that maximum, mapping them back from string keys to numbers via map(Number). Returns an empty array for empty input. Because object keys are strings, distinct numeric values that stringify identically (e.g. 1 vs "1") are conflated, and very large input arrays risk a stack overflow from the spread into Math.max.
