Converts an angle given in degrees to radians using the formula radians = degrees * Math.PI / 180, in O(1) time and space. The function returns a floating-point number, so common conversions like 180 degrees yield Math.PI exactly within double precision. It performs no input validation: non-numeric inputs follow standard arithmetic coercion and yield NaN when coercion fails, and Infinity inputs propagate to +/- Infinity. Negative degree values are supported and produce negative radian values.
