Parses a trimmed CSV string into an array of Record<string, string> rows by splitting on newlines, using the first line as the comma-separated header list, and mapping each subsequent line to an object keyed by header. Missing trailing fields default to empty strings via the nullish coalescing operator, and any header without a corresponding value yields "". The parser is naive: it splits purely on "," and "\n" with no support for quoted fields, embedded commas/newlines, or escaped quotes, and returns [] when fewer than two lines are present.
