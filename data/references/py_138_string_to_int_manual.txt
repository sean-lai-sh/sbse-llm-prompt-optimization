Manually parses a decimal integer from string s without using int(), first stripping surrounding whitespace and rejecting an empty result with ValueError. It detects an optional leading '+' or '-' sign, then accumulates digits via the Horner-style update result = result * 10 + (ord(ch) - ord('0')), raising ValueError on any non-digit character. The algorithm is O(n) in the digit count, returns the negated value when the sign was '-', and does not handle bases other than 10 or numeric Unicode digit categories beyond ASCII 0-9.
