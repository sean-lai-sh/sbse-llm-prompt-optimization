Splits the input array arr into a list of consecutive subarrays each at most size elements long by stepping the index in increments of size and slicing the array. Returns the resulting array of chunks without mutating the input, with the final chunk potentially shorter when arr.length is not a multiple of size. Throws a RangeError when size is less than or equal to 0, runs in O(n) time, and uses O(n) total space across the chunk slices.
