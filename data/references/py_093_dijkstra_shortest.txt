Implements Dijkstra's single-source shortest path algorithm on a weighted graph (dict of dict of floats) using a binary min-heap (heapq) of (distance, node) pairs and a visited set to skip stale entries. Returns a tuple (dist, prev) mapping each reachable node to its shortest distance from start and to its predecessor on that path; unreachable nodes retain distance float('inf') and prev None. Raises ValueError if start is not in the graph or if any edge has a negative weight, and supports early termination when the optional end node is popped from the heap. Time complexity is O((V + E) log V).
