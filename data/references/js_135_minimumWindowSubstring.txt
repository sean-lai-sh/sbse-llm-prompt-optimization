Finds the shortest contiguous substring of s that contains every character of t with the required multiplicities, using a sliding-window with character counts. It builds a need frequency map for t and tracks a window map plus a "have" counter equal to the number of distinct chars whose window count has reached its required value; once have equals the number of unique need keys it shrinks the window from the left while still valid, recording the smallest valid window observed. The function returns "" when no valid window exists or when s, t are missing or s is shorter than t. Time complexity is O(|s| + |t|) and space is O(|s| + |t|) for the maps.
