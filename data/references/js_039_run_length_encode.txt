Performs run-length encoding on str by walking through indices 1 to length inclusive, incrementing a count while consecutive characters match and otherwise appending the count (omitted when 1) followed by the just-ended character to the result. Returns the empty string for falsy input and runs in O(n) time and O(n) space. The output is ambiguous and not directly decodable when the source contains digits, since digit characters in the input are indistinguishable from the run-length prefixes in the output.
