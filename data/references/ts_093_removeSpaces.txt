Returns a new string with all whitespace runs in s removed by replacing the regex /\s+/g with an empty string. The \s class matches spaces, tabs, newlines, carriage returns, form feeds, and vertical tabs, but does not match Unicode whitespace such as non-breaking spaces unless the engine treats them as \s. Runs in O(n) time, leaves the original string unchanged (strings are immutable in JavaScript), and produces an empty string when the input is empty or entirely whitespace.
