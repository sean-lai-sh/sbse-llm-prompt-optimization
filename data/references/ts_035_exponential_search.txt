Locates a target in a sorted numeric array by first finding an exponentially growing bound where arr[bound] exceeds the target and then delegating to a bounded binary search over [bound/2, min(bound, n-1)]. It returns the matched index or -1 from a number[] and target number input, running in O(log i) time and O(1) space, where i is the position of the target. Empty arrays and an immediate match at index 0 are short-circuited, and the bound is capped at n - 1 so the binary search never reads past the array end.
