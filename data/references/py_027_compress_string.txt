Performs run-length encoding on a string, returning the compressed form (e.g. "aaabb" -> "a3b2") only if the result is shorter than the original; otherwise it returns the original string unchanged. It walks the input once, counting consecutive equal characters and emitting char+count when the run ends, omitting the count when it is 1. Runtime and space are O(n), an empty input is returned as-is via an early return, and the no-savings fallback ensures the function never makes the string longer.
