Sorts a numeric array in ascending order using selection sort by repeatedly finding the index of the minimum in the unsorted suffix and swapping it into the boundary position. It takes a number[] and returns a new sorted number[] in O(n^2) time for all inputs and O(n) space because of the initial shallow copy. The implementation skips the swap when the minimum is already in place, and is not a stable sort since long-distance swaps can reorder equal keys.
