Builds a rows-by-cols matrix filled with the integers 1, 2, ..., rows*cols laid out in clockwise spiral order starting from the top-left. It maintains four shrinking boundaries (top, bottom, left, right) and walks each layer in four directions, guarding the bottom and left passes to avoid overwriting cells when only a single row or column remains. The algorithm runs in O(rows * cols) time and space, and returns an empty matrix when either dimension is 0.
