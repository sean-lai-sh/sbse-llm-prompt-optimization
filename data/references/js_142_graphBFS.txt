Performs breadth-first search over an adjacency-list graph from a start node, with three modes selected by options.mode: default "traversal" returns visit order, "shortestPath" returns the node sequence to options.target via parent backtracking, and "levelOrder" returns an array of arrays grouping nodes by BFS depth. Missing start node yields [] for traversal/levelOrder and null for shortestPath. The implementation uses a JavaScript array as the queue (with shift, giving O(V) per dequeue rather than O(1)) and a Set for the visited check, so overall complexity is O(V*(V+E)) in the worst case rather than the textbook O(V+E). Note that the default traversal branch creates fresh visited/queue locals and ignores the outer ones initialized at the top of the function.
