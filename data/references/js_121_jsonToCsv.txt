Serializes an array of record objects into a CSV string with a header row built from the union of all keys observed across records. Each field is stringified (with null/undefined becoming "") and wrapped in double quotes with internal quotes doubled when it contains the delimiter, a quote, or a newline. The configurable delimiter defaults to "," and rows are joined with "\n". Non-array or empty inputs return "", and missing keys in a particular record are emitted as empty strings to keep columns aligned.
