Sorts a numeric array in ascending order using heap sort, first building a max-heap in place via a sift-down heapify routine and then repeatedly swapping the root to the end and re-heapifying the shrinking prefix. It takes a number[] and returns a new sorted number[] in O(n log n) time for all inputs and O(n) space because of the initial shallow copy plus O(log n) recursion depth in heapify. The algorithm is not stable and operates on the copy without mutating the caller's array.
