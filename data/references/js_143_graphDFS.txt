Performs recursive depth-first search over an adjacency-list graph from a start node with four modes: default "traversal" returns {order, discovery, finish} timestamps and continues into other components after start; "topological" returns a reverse post-order over all nodes; "detectCycle" uses a recursion-stack set to detect a back-edge and returns a boolean; and "allPaths" enumerates every simple path from start to options.target by backtracking the visited set. Missing start node yields [] (or false for detectCycle, [] for allPaths). All variants run in O(V+E) and rely on JavaScript's call stack, so very deep graphs may cause stack overflow. detectCycle iterates all nodes in the graph rather than only those reachable from start, treating the graph as directed.
