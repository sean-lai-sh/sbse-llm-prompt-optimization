Builds a pandas-style pivot table from a list of row dicts, grouping values by the index and columns fields and aggregating the values field with aggfunc (default is mean, returning None for empty groups). It first bucketizes raw values into a nested dict-of-dict-of-list keyed by (index_value, column_value), then applies aggfunc to each bucket to produce the final dict-of-dicts result. Rows with None in the values field are skipped from aggregation; missing index/columns fields produce a None key bucket; runs in O(n) time over input rows plus the cost of the aggregator.
