Computes the greatest common divisor of two integers a and b using the iterative Euclidean algorithm, repeatedly replacing the pair (a, b) with (b, a mod b) until b becomes 0. Returns the final value of a in O(log min(a, b)) time and O(1) space without mutating any external state. Returns the absolute value of a when b is initially 0, but does not normalize signs and so may return a negative number if either input is negative.
