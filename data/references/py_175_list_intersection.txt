Returns a list of elements that appear in both input lists, preserving the order of their first occurrence in l1 and removing duplicates. It builds a set from l2 for O(1) membership lookups and a separate seen set to filter repeats, iterating l1 once for an overall O(len(l1) + len(l2)) time and O(len(l2)) space cost. The function requires elements to be hashable and would raise TypeError on unhashable items such as lists.
