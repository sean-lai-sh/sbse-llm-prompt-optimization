A generic wrapper around JSON.parse that takes a JSON string and returns the parsed value typed as T, or null if parsing throws. It uses a try/catch to swallow SyntaxError and any other parse exceptions, providing a non-throwing alternative to the native parser. Note that the type assertion is unchecked, so the returned value is not actually validated against T at runtime, and a literal "null" string will return the value null indistinguishably from a parse failure.
