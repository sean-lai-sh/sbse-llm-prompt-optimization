Strips HTML markup from a string and decodes a small set of common HTML entities, returning the cleaned, trimmed text. It first removes anything matching <[^>]+> via regex, then applies separate regex substitutions to convert &amp;, &lt;, &gt;, &nbsp;, and &quot; to their literal characters before stripping leading/trailing whitespace. The implementation is O(n) per pass, does not handle numeric entities (&#39;, &#x27;) or other named entities, and the simple tag regex can be fooled by '>' characters appearing inside attribute values.
