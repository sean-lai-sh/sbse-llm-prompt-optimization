Rotates a square n by n number matrix 90 degrees clockwise into a freshly allocated n by n matrix. It allocates the result with Array.from and a fill of zeros, then copies each source cell at (r, c) to position (c, n - 1 - r) using a double loop, running in O(n^2) time and O(n^2) extra space. Returns an empty array for an empty input and assumes the matrix is square; a non-square input would produce an undefined-shaped result.
