Finds the first pair of indices in nums whose values sum to target using a single-pass hash map approach in O(n) time and O(n) space. For each element it checks whether target minus the current value already exists in the seen Map, recording the pair if so, then stores the current index keyed by its value (overwriting any earlier index for duplicates). It collects all matches into a results array but only returns the first pair found, or null when no pair exists. The function never pairs an index with itself because the map lookup happens before the current element is inserted.
