Finds and returns the longest palindromic contiguous substring of s using the expand-around-center technique, which considers both odd-length centers (i, i) and even-length centers (i, i+1) at every index. For each center it walks outward while characters match and updates the best (start, end) bounds whenever a longer palindrome is found, running in O(n^2) time and O(1) extra space. The function returns the empty string for empty input and, on ties, returns the first longest palindrome encountered during the left-to-right scan.
