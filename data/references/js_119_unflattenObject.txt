Inverts a flattened key/value object by splitting each key on the supplied separator (default ".") and rebuilding the nested object structure. For every entry it walks the path, creating intermediate plain objects where needed and overwriting any non-object placeholder it encounters mid-path. The terminal segment receives the entry's value. Non-object or null inputs return {}; the function does not produce arrays for numeric segments and silently overwrites conflicting values when one key is a prefix of another.
