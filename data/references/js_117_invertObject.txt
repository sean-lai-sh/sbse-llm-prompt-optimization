Produces a new object that swaps the keys and values of the input, stringifying each value to use as the new key. When multiple original keys map to the same stringified value, the function collects them into an array (single second occurrence becomes a 2-element array, then each subsequent collision is appended). Non-object or null inputs return {}. Because values are coerced via String(), distinct values like 1 and "1" or true and "true" collide on the same key, and object-valued entries become "[object Object]".
