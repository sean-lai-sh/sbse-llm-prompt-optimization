Truncates a string to at most max_length characters, appending an ellipsis (default '...') when truncation occurs, and returns the resulting string. If the input already fits within max_length it is returned unchanged; if max_length is smaller than or equal to the ellipsis length the function returns just a prefix of the ellipsis itself. Otherwise it slices text to (max_length - len(ellipsis)) characters and appends the ellipsis so the final length equals max_length, running in O(max_length) time.
