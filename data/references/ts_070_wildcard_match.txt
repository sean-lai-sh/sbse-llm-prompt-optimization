Performs wildcard pattern matching of s against p where '?' matches any single character and '*' matches any (possibly empty) sequence, using a bottom-up (m+1) by (n+1) boolean DP table. The first row is initialized so leading '*' characters can match the empty prefix, and each cell expresses '*' as dp[i][j-1] (match empty) or dp[i-1][j] (consume one more character), while '?' or a literal match consumes one character via dp[i-1][j-1]. Time and space complexity are O(m*n), and the function returns dp[m][n].
