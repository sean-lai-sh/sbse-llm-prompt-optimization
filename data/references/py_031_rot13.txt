Implements the ROT13 substitution cipher on a text string and returns the transformed string. For each ASCII letter it computes (ord(ch) - base + 13) % 26 + base where base is 'A' or 'a' depending on case, leaving non-alphabetic characters unchanged. Runtime is O(n) with O(n) space; the transformation is its own inverse (applying it twice yields the original), and Unicode letters outside ASCII are passed through unmodified.
