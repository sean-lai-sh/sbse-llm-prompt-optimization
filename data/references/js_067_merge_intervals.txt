Merges a list of [start, end] intervals into the minimal set of non-overlapping intervals by first sorting a shallow copy by start time and then sweeping left-to-right, extending the last merged interval's end whenever the next interval's start is less than or equal to it. Overall complexity is O(n log n) due to the sort, with O(n) extra space for the output. The first interval is pushed by reference and mutated via last[1] = Math.max(...) on overlap, while non-overlapping intervals are pushed as cloned arrays via spread. It returns an empty array for null/undefined or empty input and treats touching intervals (current[0] === last[1]) as overlapping.
