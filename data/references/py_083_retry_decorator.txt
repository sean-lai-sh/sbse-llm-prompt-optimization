A decorator factory that wraps a function so it is retried up to max_attempts times when it raises any exception in the configured exceptions tuple, sleeping current_delay seconds between attempts and multiplying the delay by backoff after each failure. An optional on_failure callback receives the exception and 1-based attempt number after each failed try and is itself wrapped in try/except so callback errors do not break the retry loop. Validates that max_attempts >= 1, delay >= 0, and backoff >= 1 at decoration time, and re-raises the last exception if all attempts fail; uses functools.wraps to preserve metadata.
