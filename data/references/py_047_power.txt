Computes base raised to the integer exp using exponentiation by squaring and returns the result as a float. It returns 1.0 for exp == 0, recurses with the negated exponent and reciprocates the result for negative exp, and otherwise iteratively multiplies result by base whenever the current low bit of exp is 1 while squaring base and right-shifting exp via integer division. Time complexity is O(log |exp|) with O(1) iterative space (one recursive call for negatives), and the function handles negative exponents and zero exponent correctly but does not guard against base == 0 with exp < 0 (which would raise ZeroDivisionError).
