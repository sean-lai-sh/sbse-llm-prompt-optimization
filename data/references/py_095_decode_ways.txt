Counts the number of ways to decode a digit string s under the A=1...Z=26 mapping, returning an int. It validates that every character is a digit (raising ValueError otherwise) and uses a 1D DP array where dp[i] sums dp[i-1] (when the i-th digit is non-zero) and dp[i-2] (when the two-digit suffix is between 10 and 26 inclusive). Returns 0 for an empty input or any string starting with '0' (since dp[1] is set to 0 in that case); runs in O(n) time and space and the memo parameter is accepted for API symmetry but not actually used.
