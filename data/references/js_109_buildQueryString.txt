Serializes a plain object of parameters into a URL query string with a leading "?". It iterates the object's own entries, URL-encoding keys and values via encodeURIComponent and emitting one "key=value" pair per element of array values, a bare "key" for boolean true, and skipping null/undefined entries. Non-object or null inputs return an empty string, as does an object that produces no parts. The output is "?" joined by "&" between parts, with no special handling for nested objects or false values (false is serialized as "key=false").
