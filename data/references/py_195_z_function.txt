Finds all starting indices of pattern within text by computing the Z-function over the concatenated string pattern + "$" + text and scanning for positions where the Z-value equals the pattern length. The Z-array is computed in linear time via the standard window [lo, hi] technique that reuses previously matched lengths via the mirror index, then offset = m + 1 is used to align with positions inside text. Total runtime is O(n + m) with O(n + m) extra space; empty text or pattern short-circuits to an empty list, and the "$" sentinel ensures matches cannot straddle the boundary.
