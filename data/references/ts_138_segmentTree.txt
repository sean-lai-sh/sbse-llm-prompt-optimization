Builds a sum-aggregating segment tree over the nums array stored in a 4n-sized flat array indexed from 1, and returns update and query closures. The recursive build assigns each leaf to a single nums element and each internal node to the sum of its two children; point update walks down to the matching leaf, sets the new value, and propagates the new sum back up; range query recurses on overlapping segments and returns 0 for fully disjoint ranges and the cached node sum for fully covered ones. Build is O(n) and both update and query are O(log n); update also writes the new value back into the original nums array, mutating it.
