Invokes an asynchronous function repeatedly with exponential backoff until it succeeds or the retry budget is exhausted. It accepts the async function fn, a retries count (default 3), and a base delayMs (default 100), returning the resolved value of fn or rethrowing the last caught error. Between attempts it waits delayMs * 2^attempt milliseconds, performing up to retries+1 total invocations. There is no jitter, no per-error filtering, and no abort signal, so non-retryable errors are still retried for the full budget.
