Recursively produces a deep copy of a value, dispatching on type to handle primitives, null, arrays, Date, Map, Set, and plain objects while preserving the original prototype via Object.create. It is generic over T, taking a T and returning a T in O(n) time and O(n) space proportional to the total node count, with recursion depth equal to the structure's depth. The implementation does not detect cyclic references, so circular structures cause infinite recursion, and it does not copy non-enumerable properties, symbol keys, getters/setters, or class instances with private state.
