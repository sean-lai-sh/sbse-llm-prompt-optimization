Returns the items of a dictionary as a list of (key, value) tuples sorted in ascending order by value using Python's stable Timsort. It is a thin wrapper over sorted(d.items(), key=lambda kv: kv[1]) and runs in O(n log n) time and O(n) space. The function will raise TypeError if the values are not mutually comparable, and equal values keep their original insertion order due to sort stability.
