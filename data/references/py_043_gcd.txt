Computes the greatest common divisor of two integers a and b using the iterative Euclidean algorithm and returns the absolute value of the result. The loop repeatedly replaces (a, b) with (b, a % b) until b becomes zero, then returns abs(a) so the answer is non-negative regardless of input sign. Time complexity is O(log min(|a|, |b|)) with O(1) space, and the function correctly handles zero inputs because gcd(x, 0) = |x|.
