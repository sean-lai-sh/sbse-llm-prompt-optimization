Sorts a list of integers in ascending order using the insertion sort algorithm and returns the same list. For each index from 1 onward it stores the current element as a key and shifts any preceding elements that are larger one position to the right, then drops the key into the resulting gap. The algorithm runs in O(n^2) worst-case time and O(1) auxiliary space, mutates the input in place, is stable, and degrades gracefully to O(n) on nearly-sorted input.
