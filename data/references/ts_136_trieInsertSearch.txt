Constructs a prefix-tree (Trie) seeded with an array of words and exposes insert, search, and startsWith closures over a shared root node whose children are stored in a Map keyed by character. insert walks or creates child nodes per character then marks the terminal node's isEndOfWord, search walks existing children and returns true only when the final node is marked end-of-word, and startsWith walks children and returns true as long as the prefix path exists. All three operations run in O(L) time per call where L is the length of the input string, and the constructor pre-inserts every supplied word.
