Returns the longest string prefix shared by every entry in the input array strs by initializing prefix to strs[0] and trimming one character from its end until each subsequent string starts with it. Returns the empty string when strs is null, empty, or when no common prefix exists. Runs in O(S) time where S is the sum of all character comparisons performed, bounded by total characters across the inputs, and uses O(m) space for the prefix where m is the length of the shortest matching prefix.
