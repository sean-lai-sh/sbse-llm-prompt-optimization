Computes the length of the longest common subsequence of two strings s1 and s2 using a classic (m+1) x (n+1) dynamic programming table in O(m*n) time and space, returning a tuple (length, sequence_string). When return_sequence is False the second tuple element is an empty string; when True the function reconstructs one valid LCS by walking the table from (m, n) back toward (0, 0), preferring diagonal moves on character matches and otherwise stepping toward the larger of the up/left neighbor. Returns (0, '') immediately if either input string is empty.
