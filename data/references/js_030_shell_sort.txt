Sorts a shallow copy of arr in ascending order using Shell sort with the original Shell gap sequence, halving the gap each outer iteration and performing gapped insertion sort within each pass. Returns the sorted copy without mutating the input. Time complexity depends on the gap sequence; with this halving sequence it averages roughly O(n^2) and is bounded by O(n^2) in the worst case, while using O(n) extra space for the copy and O(1) auxiliary space for sorting.
