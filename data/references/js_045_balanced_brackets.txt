Checks whether the round, square, and curly brackets in the input string str are properly matched and nested using a stack-based scan that pushes openers and pops the matching opener for each closer via a lookup table. Returns false on a mismatched closer or one that appears with an empty stack, and returns true only when the stack is empty after processing every character. Runs in O(n) time and O(n) auxiliary space, ignoring all non-bracket characters and treating an empty input as balanced.
