Performs meta (one-sided) binary search on a sorted list of integers, building the answer index bit by bit from the most significant bit downward. Starting with pos=0, for each bit it tentatively sets that bit via pos | (1 << i) and accepts the new position only if it stays in bounds and arr[new_pos] <= target, ultimately returning pos if it indexes the target or None otherwise. Time complexity is O(log n) with O(1) space, empty input is handled by an early return, and the technique avoids multiplication/division in the index computation.
