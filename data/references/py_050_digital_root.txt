Computes the digital root of an integer n by repeatedly summing its decimal digits until a single-digit value remains. It first takes abs(n) so negative inputs are treated as their absolute value, then while n >= 10 it converts n to a string, sums the integer values of its characters, and reassigns to n. Time complexity is roughly O(log n) per iteration over a logarithmic number of iterations; the function returns n unchanged when it is already a single digit (including 0).
