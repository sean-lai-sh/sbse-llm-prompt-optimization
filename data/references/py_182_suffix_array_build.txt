Builds the suffix array of string s, returning a list of starting indices that order all suffixes of s lexicographically. It uses prefix-doubling with rank pairs (rank[i], rank[i+k]) as the sort key and a length-doubling outer loop, collapsing equal pairs into the same rank after each pass and terminating early when all ranks become unique. The implementation runs in O(n log^2 n) time due to using Python's generic sort each round (instead of radix sort) with O(n) auxiliary space, and returns an empty list for an empty string.
