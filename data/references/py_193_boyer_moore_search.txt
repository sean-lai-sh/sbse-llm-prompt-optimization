Searches text for all non-overlapping occurrence start indices of pattern using the full Boyer-Moore algorithm with both the bad-character and good-suffix heuristics. It precomputes the bad_char table mapping each pattern character to its rightmost index and a good_suffix shift table built from a border array, then aligns the pattern against the text and on each mismatch advances by max(bad_char_shift, good_suffix_shift). Empty text or pattern, or a pattern longer than the text, returns an empty list; preprocessing is O(m + sigma) and the search itself runs in worst-case O(n) for typical inputs.
