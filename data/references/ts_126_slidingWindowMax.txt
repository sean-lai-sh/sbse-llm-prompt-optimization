Computes the maximum value in every contiguous window of size k over the nums array using a monotonic decreasing deque that stores indices. For each new index it pops indices that have fallen out of the window from the front and pops smaller values from the back before pushing, then once the window is fully formed it records nums at the deque's front as the current max. Runs in amortized O(n) time and O(k) extra space, returning [] when nums is empty or k <= 0; note the use of Array#shift makes the front-pop technically O(k) per call.
