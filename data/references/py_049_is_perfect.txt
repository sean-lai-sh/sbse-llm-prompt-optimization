Returns True if the positive integer n is a perfect number (equal to the sum of its proper divisors) and False otherwise. It seeds the divisor sum with 1 and iterates i from 2 while i*i <= n, adding both i and n // i whenever i divides n (skipping the duplicate when i == n // i, i.e. n is a perfect square). Time complexity is O(sqrt(n)) with O(1) space; values less than 2 are rejected by an early return so 0, 1, and negatives correctly return False.
