Builds a segment tree over an integer array and processes a sequence of ("sum", lo, hi) range-sum and ("update", idx, val) point-update queries, returning the answers to the sum queries in order. The tree is stored in a flat array of size 4n with recursive build, update, and query helpers that each traverse O(log n) nodes per call, mutating both the tree and the underlying nums on updates. Empty input arrays return an empty result list, and the function silently ignores any query whose op tag is neither "sum" nor "update".
