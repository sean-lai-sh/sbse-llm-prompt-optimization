Computes the minimum number of coins from coins needed to make amount using bottom-up DP where dp[i] is the fewest coins summing to i, alongside a coin_used array for reconstruction. Returns a tuple (count, coins_list) where count is -1 and coins_list is None when amount is unreachable, and coins_list is None unless return_coins is True (in which case it backtracks through coin_used to list the chosen denominations). Raises ValueError for negative amount, empty coins, or non-positive denominations; runs in O(amount * len(coins)) time and O(amount) space.
