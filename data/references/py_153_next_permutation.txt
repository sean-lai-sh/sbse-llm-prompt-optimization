Computes the lexicographically next permutation of an integer list and returns it as a new list, leaving the original input untouched by copying it up front. It scans from the right to find the first index i where a[i] < a[i+1], swaps a[i] with the smallest value to its right that is still greater than a[i], then reverses the suffix after i to put it in ascending order. If the input is already the largest permutation (strictly decreasing), the algorithm reverses the whole array, yielding the smallest permutation; it runs in O(n) time and O(n) extra space for the copy.
