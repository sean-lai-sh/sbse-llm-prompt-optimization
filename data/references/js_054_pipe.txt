Composes a variadic list of unary functions into a single left-to-right pipeline using Array.prototype.reduce, so pipe(f, g, h)(x) evaluates as h(g(f(x))). When called with no functions it returns the identity function x => x, gracefully handling the empty case. The returned function accepts a single seed value and threads it through each fn in declaration order, with O(n) time in the number of functions. It does not propagate this, support multi-argument first calls, or await Promises between stages.
