Returns the cube of the numeric input n by evaluating n * n * n in O(1) time and space. Sign is preserved because cubing an odd power keeps the original sign, and floating-point inputs are supported via IEEE 754 multiplication. The function does no input validation: non-numeric arguments propagate JavaScript's coercion rules and yield NaN when coercion fails. Magnitudes large enough to overflow doubles return Infinity or -Infinity.
