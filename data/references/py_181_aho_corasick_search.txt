Implements multi-pattern string search via the Aho-Corasick automaton, returning a dict mapping each input pattern to the list of starting indices where it occurs in text. It first builds a goto trie of all patterns, then computes failure links and merged output sets via a BFS over depth-1 states, and finally scans text in a single pass following goto/fail transitions and recording every output match. The construction is O(sum of pattern lengths) and the search adds O(len(text) + total matches), with patterns that never occur mapping to empty lists; a small guard in fail-link construction prevents self-loops by resetting fail[s] to 0 when it would point at itself.
