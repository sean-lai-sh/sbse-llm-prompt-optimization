Rotates a square n by n matrix 90 degrees clockwise in place and returns the same matrix reference. The algorithm first transposes the matrix by swapping matrix[i][j] with matrix[j][i] for j > i, then reverses each row using Array.prototype.reverse. It runs in O(n^2) time and O(1) extra space. Empty matrices are returned unchanged, and the function assumes the input is square; non-square inputs will produce incorrect results since the swap loop skips off-diagonal elements that don't exist.
