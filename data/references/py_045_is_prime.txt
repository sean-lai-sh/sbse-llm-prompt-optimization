Returns True if the integer n is prime and False otherwise using trial division. It rejects values < 2, returns True for 2, rejects all other even numbers, then tests odd divisors from 3 up to floor(sqrt(n)) inclusive, returning False on the first divisor found. Time complexity is O(sqrt(n)) with O(1) space; the sqrt-bound and odd-only stride make it roughly twice as fast as naive trial division, and it is deterministic but slow for very large n.
