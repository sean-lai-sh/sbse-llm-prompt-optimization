Counts the ASCII vowels (both cases of a, e, i, o, u) in the input string s by running a global regex match and returning the resulting array's length, defaulting to an empty array via the nullish-coalescing operator when no matches are found. Runs in O(n) time over the string. It does not count 'y', accented vowels, or non-Latin vowel characters, so it is purely an English ASCII heuristic.
