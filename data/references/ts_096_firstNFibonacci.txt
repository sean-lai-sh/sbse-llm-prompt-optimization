Generates the first n Fibonacci numbers iteratively starting from [0, 1], appending each subsequent term as the sum of the two prior entries; runs in O(n) time and O(n) space for the output array. Returns an empty array when n is zero or negative and the single-element [0] when n is 1, otherwise returns an array of length n. Because each term is stored as a JavaScript number, precision is lost beyond 2^53 (around the 79th Fibonacci number), at which point values become approximate.
