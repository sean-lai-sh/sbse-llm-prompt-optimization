Performs an iterative binary search for target within the sorted array arr, maintaining inclusive low and high bounds and computing the midpoint with Math.floor of their sum. Returns the index of the first match found or -1 when the value is absent. Runs in O(log n) time and O(1) space, requires arr to be sorted in ascending order with values comparable via <, and could overflow the midpoint calculation only for arrays exceeding JavaScript's safe integer range.
