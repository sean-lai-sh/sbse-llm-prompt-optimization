Inserts each value of the input list into a self-balancing AVL binary search tree and returns the resulting in-order traversal as a sorted list of unique values. The internal recursive insert function tracks node heights, computes balance factors, and applies the four standard AVL rotations (LL, RR, LR, RL) to keep the tree height bounded by O(log n). Duplicate values are silently ignored, an empty input yields an empty list, and the overall cost is O(n log n) time with O(n) space for the tree nodes.
