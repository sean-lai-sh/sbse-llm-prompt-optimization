Safely resolves a dot-delimited property path against an object, returning a default value when any segment is missing or traverses into a non-object. Inputs are the source object, a string path like "a.b.c", and an optional defaultValue (undefined by default); the output is the resolved value or the default. It iterates the split path in O(depth) time, returning the default if the current value becomes null/undefined or non-object mid-traversal, or if the final lookup is undefined. Note that it splits only on "." and does not support bracket notation, escaped dots, or array indices distinct from object keys.
