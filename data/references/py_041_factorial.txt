Computes n! iteratively for a non-negative integer n and returns the result as an int. It initializes result to 1 and multiplies it by every integer from 2 through n in a for-loop, returning 1 for n == 0 or n == 1 since the loop body does not execute. The function raises ValueError for negative n, runs in O(n) multiplications, uses O(1) auxiliary space, and benefits from Python's arbitrary-precision ints so it does not overflow for large arguments.
