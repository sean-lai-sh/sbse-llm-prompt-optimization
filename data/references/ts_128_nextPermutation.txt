Computes the lexicographically next permutation of the input number array using the classic in-place algorithm applied to a shallow copy. It scans from the right for the first index i where arr[i] < arr[i+1], swaps arr[i] with the rightmost element greater than it, then reverses the suffix after i to yield the smallest tail; when no such i exists (the array is fully descending) it just reverses the whole array, wrapping back to the smallest permutation. Runs in O(n) time and O(n) space due to the spread copy, and the original input array is not mutated.
