Wraps fn in a memoizing closure backed by a Map whose keys are JSON.stringify of the argument array, returning the cached result on subsequent calls with equivalent arguments. The cache grows unboundedly for the lifetime of the returned function, giving O(1) average lookup but O(n) memory in the number of distinct argument tuples. Because keys are produced via JSON.stringify, arguments containing functions, undefined, symbols, circular references, or values that serialize identically (e.g. distinct objects with the same shape) will collide or throw. The wrapper preserves this via fn.apply but does not differentiate calls by receiver.
