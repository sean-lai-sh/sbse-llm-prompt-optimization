Sorts a numeric array in ascending order using Shell sort with a halving gap sequence starting at floor(n / 2) and dividing by two each pass. It accepts a number[] and returns a new sorted number[] in O(n^2) worst-case time but typically much better, with O(n) space because the input is shallow-copied first. The inner loop is a gapped insertion sort, and the algorithm is not stable due to the long-distance moves between elements.
