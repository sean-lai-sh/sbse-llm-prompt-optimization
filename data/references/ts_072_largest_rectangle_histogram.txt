Computes the area of the largest axis-aligned rectangle that fits inside a histogram described by heights using a monotonic increasing stack of indices in O(n) time and O(n) space. It iterates one position past the end and treats that position as a sentinel height of 0 to flush all remaining bars; whenever the current height drops below the stack's top, it pops bars and computes their maximal rectangle width as i - stack_top - 1 (or i if the stack is empty). Returns the maximum area encountered, which is 0 for an empty input.
