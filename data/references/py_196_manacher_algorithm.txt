Runs Manacher's algorithm to compute, for every position in the input string, the radius of the longest palindrome centered there, returning these radii as a list of length len(s). It transforms s by interleaving '#' separators so that even- and odd-length palindromes can be handled uniformly, then expands palindromes around each center while exploiting symmetry around the rightmost known palindrome (center, right) for amortized O(n) total work. The function returns an empty list for an empty input string, and although it internally also tracks the longest palindromic substring, only the per-character radius array is returned.
