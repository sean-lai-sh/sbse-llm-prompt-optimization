Determines whether the integer n is a positive power of two using the classic bit trick n > 0 && (n & (n - 1)) === 0, which holds exactly when n has a single set bit. The check runs in O(1) time and correctly rejects zero and negative numbers. Because JavaScript bitwise operators coerce operands to 32-bit signed integers, the function only works reliably for n within the int32 range and may produce incorrect results for very large numbers or non-integer inputs.
