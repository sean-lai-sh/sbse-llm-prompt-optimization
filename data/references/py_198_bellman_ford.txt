Computes shortest-path distances from source to every node of an undirected weighted graph using Bellman-Ford, returning a dict mapping node id to distance or None if a negative-weight cycle is detected. It performs up to num_nodes - 1 relaxation passes over the edge list, treating each edge as bidirectional by attempting relaxation in both (u, v) and (v, u) directions, with an early exit when a pass makes no updates. After relaxation it does one more pass that returns None on any further improvement, signaling a negative cycle reachable from source; runtime is O(V * E).
