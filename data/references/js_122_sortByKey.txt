Returns a new array sorted by a single object property, leaving the input array untouched via slice() before calling Array.prototype.sort. Direction defaults to "asc" and is inverted to descending when "desc" is passed. Null and undefined values are pushed to the end regardless of direction, and equal values preserve relative order subject to the engine's sort stability. Non-array inputs short-circuit to []; comparison uses JavaScript's < operator, so it relies on the natural ordering of the underlying property type and may produce surprising results for mixed-type values.
