Counts the number of digits in n by stringifying its absolute value, removing a single decimal point if present via replace(".", ""), and returning the resulting string length. Runs in O(d) time and O(d) space where d is the digit count. Math.abs ensures the sign character is never counted, and stripping the dot lets the function count digits in floats (e.g. 12.34 yields 4). For very large numbers that JavaScript stringifies in scientific notation (e.g. 1e21), the count includes the 'e' and exponent characters, producing an incorrect result; NaN and Infinity yield the lengths of those literal strings (3 and 8 respectively).
