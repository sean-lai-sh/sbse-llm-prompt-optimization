Sorts a copy of the input array arr in ascending order using the bubble sort algorithm with an early-termination optimization that breaks once a pass completes without swaps. Operates on a shallow clone created via spread, so the original array is not mutated, and returns the sorted copy. Worst- and average-case time is O(n^2) with O(n) best-case on already sorted input, using O(n) extra space for the copy and ordering values via the > operator.
