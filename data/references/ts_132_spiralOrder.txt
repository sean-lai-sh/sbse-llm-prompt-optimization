Traverses a 2D matrix in clockwise spiral order and returns the visited values as a flat array. It maintains four shrinking boundaries (top, bottom, left, right) and each iteration emits the top row left-to-right, the right column top-to-bottom, optionally the bottom row right-to-left, and optionally the left column bottom-to-top, with the inner two passes guarded to avoid double-visiting on degenerate single-row or single-column remainders. Runs in O(rows * cols) time and supports non-square matrices, returning an empty array when the matrix has no rows.
