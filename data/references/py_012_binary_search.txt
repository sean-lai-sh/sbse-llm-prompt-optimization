Performs an iterative binary search on a sorted list of integers for the given target, returning the matching index or None if not found. It maintains low and high pointers, computes the midpoint with integer division, and narrows the window left or right depending on how the midpoint element compares to the target. Time complexity is O(log n) with O(1) auxiliary space, the input must be sorted in ascending order, and an empty list returns None on the first loop check.
