Flattens a nested object into a single-level object whose keys are dot-delimited (or custom-separator) paths to leaf values. It walks the input recursively, prefixing each key with the accumulated path and copying non-object leaves directly while recursing into plain object values. Arrays and null are treated as leaf values and copied verbatim rather than recursed into. Non-object or null root inputs return {}, and the depth of recursion is bounded by the nesting depth of the input.
