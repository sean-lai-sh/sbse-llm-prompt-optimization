Returns the single character that occurs most often in the input string str by tallying counts in a plain object via a for-of iteration and then scanning the entries for the maximum count. Returns null for falsy input such as empty string, null, or undefined. Runs in O(n) time and O(k) space where k is the alphabet size; iterates by code point rather than UTF-16 unit but ties are broken by Object.entries iteration order, which is insertion order for string keys.
