Combines a variable number of input arrays into a single array of tuples by zipping element-wise up to the length of the shortest input array. Computes minLen via Math.min over all input lengths and then for each index produces an array containing the i-th element of every input. Runs in O(k * minLen) time and O(k * minLen) space where k is the number of arrays, and returns an empty array when called with no arguments because Math.min() of zero arguments yields Infinity but the loop is bounded by minLen which becomes Infinity, making this caller-error edge case worth noting.
