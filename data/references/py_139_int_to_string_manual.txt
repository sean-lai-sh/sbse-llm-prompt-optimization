Converts integer n into its base-10 string representation without using str(), short-circuiting to "0" when n is zero. It records the sign, takes the absolute value, then repeatedly extracts the last digit via n % 10 (mapped to its ASCII character with chr(ord('0') + d)) and floor-divides n by 10 until n becomes zero, finally prepending '-' when negative and reversing and joining the collected digits. Runs in O(d) time and space where d is the number of digits, and only supports integer input in base 10.
