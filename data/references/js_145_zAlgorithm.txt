Performs pattern matching by constructing the concatenated string pattern + "$" + text and computing its Z-array, where positions whose Z-value equals the pattern length correspond to matches in text. The Z-array is built in linear time using the classic [l, r] window optimization that copies values from a previously computed prefix when the current index lies inside the window before extending naively. The function returns {matches, zArray, concatZArray}, where zArray is the Z-array of text alone (computed by an inner computeZ helper) and concatZArray is the one over the concatenation. Empty pattern or text yields {matches: [], zArray: []}; total time is O(n + m) and extra space is O(n + m).
