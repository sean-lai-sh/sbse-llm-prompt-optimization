Returns a new array containing the elements of arr in reverse order, in O(n) time and O(n) extra space. It first calls slice() to produce a shallow copy and then mutates that copy in place via Array.prototype.reverse, leaving the original arr untouched (non-mutating from the caller's perspective). Element references are copied, so nested objects are still shared between the original and the returned array. Throws a TypeError if arr is null or undefined and treats sparse arrays per slice/reverse semantics, including preserving holes.
