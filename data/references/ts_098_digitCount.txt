Returns the number of digit characters in n by stringifying its absolute value, stripping a single decimal point, and taking the resulting string's length, running in O(d) time. The redundant ternary n === 0 ? 0 : n is a no-op since Math.abs(0) is already 0. The function counts both integer and fractional digits combined when n is non-integer, but it does not handle exponential notation, leading zeros (which numbers do not store), or BigInt inputs, and negative signs are removed by Math.abs.
