Builds an uppercase abbreviation/acronym from a phrase by concatenating the first character of each whitespace-delimited word. It strips and splits the input on whitespace, takes word[0] for each non-empty token, uppercases it, and joins the letters into a single string. Runtime is O(n) in the input length; it gracefully handles extra surrounding whitespace and the empty input (returning ''), but does not skip stop-words and uses the first character even if it is a digit or punctuation.
