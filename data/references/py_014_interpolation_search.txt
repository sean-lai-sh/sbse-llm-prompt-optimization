Performs interpolation search on a sorted list of integers for a target, returning its index or None if not present. Instead of always picking the midpoint it estimates the probable position via linear interpolation between arr[low] and arr[high], then narrows the search window like binary search based on the comparison. Average-case time on uniformly distributed data is O(log log n) but worst case is O(n); a guard handles the arr[low] == arr[high] case to avoid division by zero, and the bounds check arr[low] <= target <= arr[high] short-circuits when the target is outside the current range.
