Computes the length of the longest run of consecutive integers in the input array using a Set for O(1) membership tests. It iterates the set and only begins counting when the current value has no predecessor (num - 1 absent), then walks num + 1, num + 2, ... while present, tracking the maximum streak length. The skipping rule keeps the total work amortized O(n) regardless of duplicates or unsorted ordering, and the function returns 0 for empty input.
