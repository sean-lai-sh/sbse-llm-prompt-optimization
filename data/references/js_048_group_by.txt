Groups elements of the input array arr into buckets within a plain object using keyFn, which may be either a function applied to each item or a property name to look up on the item. Returns the resulting object whose values are arrays of elements sharing each derived key, built via reduce starting from an empty object. Runs in O(n) time and O(n) space, preserves the original order of items within each group, and coerces keys to strings as is standard for object property names.
