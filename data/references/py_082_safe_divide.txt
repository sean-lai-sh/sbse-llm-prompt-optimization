Performs numerator/denominator division with extensive guards: it raises TypeError for non-numeric arguments, returns default (or raises ValueError when raise_on_zero is True) for NaN inputs, and short-circuits to 0.0 when the denominator is infinite. Division by zero returns default by default or raises ZeroDivisionError when raise_on_zero is True; an infinite result similarly returns default or raises OverflowError. When precision is provided the successful result is rounded to that many decimal places via round(); the function runs in O(1) time.
