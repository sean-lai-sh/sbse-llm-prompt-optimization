Returns true if and only if n is a finite JavaScript number whose value equals its floor, in O(1) time and space. The check combines typeof n === "number" (which excludes BigInts, strings, and other types), Number.isFinite (which rejects Infinity, -Infinity, and NaN), and Math.floor(n) === n (which rules out non-integer fractional values). It is functionally equivalent to Number.isInteger for typical inputs, including correctly handling -0 and very large but representable integers within Number.MAX_SAFE_INTEGER bounds. Numeric strings such as "5" return false because of the typeof guard.
