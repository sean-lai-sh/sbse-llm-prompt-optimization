Converts a snake_case string str into camelCase by first lowercasing the entire input and then replacing each underscore-followed-by-lowercase-letter pair with the uppercase version of that letter using a regex replacement callback. Returns a new string in O(n) time and O(n) space without mutating the input. Trailing underscores or underscores not followed by a-z are left intact, and the first character is never capitalized so the result is lowerCamelCase rather than PascalCase.
