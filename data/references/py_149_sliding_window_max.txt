Computes the maximum value of every contiguous window of size k in nums using a monotonic deque of indices that stores candidates in decreasing order of their corresponding values. For each new index it pops indices from the front whose position has fallen out of the window and pops indices from the back whose values are smaller than the current value, then appends the current index and, once the first full window is reached, records nums[dq[0]] as the window maximum. The amortized complexity is O(n) time and O(k) space, the function returns an empty list when nums is empty or k is 0, and produces n - k + 1 results otherwise.
