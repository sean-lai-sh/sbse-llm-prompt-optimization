Finds the two indices in nums whose values sum to target using a single pass with a Map from value to its earliest index, achieving O(n) time and O(n) space. For each element it checks whether target minus the current value has already been seen, returning the stored index together with the current index as a tuple [j, i] with j less than i. The function assumes exactly one solution exists and throws an Error when none is found; it does not handle the case of using the same index twice because the lookup happens before insertion.
