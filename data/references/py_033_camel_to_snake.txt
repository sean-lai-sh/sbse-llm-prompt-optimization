Converts a camelCase or PascalCase identifier to snake_case using two regex substitutions. The first inserts an underscore before an uppercase letter that is followed by lowercase letters (handling sequences like "HTTPServer" -> "HTTP_Server"), and the second inserts an underscore between a lowercase or digit character and a following uppercase letter, after which the whole string is lowercased. The function returns a new string in O(n) time, correctly handles acronym boundaries, and leaves strings without uppercase letters unchanged.
