Merges a list of (start, end) integer interval tuples by optionally sorting them by (start, end) and then sweeping left to right, extending the last merged interval whenever the next interval's start lies within or touches it (current_start <= last_end). Returns a new list of non-overlapping intervals; treats touching intervals (e.g. (1,3) and (3,5)) as overlapping and merges them. Validates that no interval has start > end (raising ValueError), returns [] for empty input, and runs in O(n log n) when sort_first is True or O(n) when the caller guarantees sorted input.
