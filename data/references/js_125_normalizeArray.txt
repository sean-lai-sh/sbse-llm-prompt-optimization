Min-max normalizes a numeric array into the [0, 1] range and returns a new array of the same length. It computes the min and max via Math.min/Math.max with spread, then maps each element to (n - min) / (max - min). Non-array or empty inputs return []; when min equals max (all values identical) it returns an array of zeros to avoid division by zero. The Math.min/max spread can hit the engine's argument-count limit and stack-overflow on very large arrays.
