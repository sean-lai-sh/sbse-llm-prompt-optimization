Decodes a run-length-encoded sequence of (character, count) tuples back into the original string by multiplying each character by its count and joining the parts. Uses a list accumulator with a single "".join at the end to avoid the quadratic cost of repeated string concatenation, giving O(N) time where N is the total decoded length. Returns the empty string for an empty input list and produces empty fragments for tuples whose count is zero or negative.
