Searches a sorted numeric array using jump search, advancing in fixed blocks of size floor(sqrt(n)) until it overshoots the target, then performing a linear scan within the last block. It takes a number[] and a target number and returns the matching index or -1, running in O(sqrt(n)) time and O(1) space. The array must be sorted in ascending order, and an empty input is short-circuited with -1; the inner scan caps its upper bound at n to avoid out-of-bounds reads.
