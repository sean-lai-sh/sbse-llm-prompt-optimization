Reports whether the input n is an integer by delegating directly to the built-in Number.isInteger predicate, returning true only for finite numeric values with no fractional component. The check is O(1) and unlike the global isFinite/parseInt-based alternatives it does not coerce its argument, so NaN, Infinity, -Infinity, and any non-number type produce false. The function adds no validation beyond what Number.isInteger already provides.
