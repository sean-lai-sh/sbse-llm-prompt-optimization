Performs run-length encoding over a 2D matrix flattened in row-major order, returning a list of (value, count) tuples for each maximal run of equal entries. It first concatenates all rows into a single flat list, then iterates once tracking the current value and count, emitting a tuple whenever the value changes and a final tuple at the end. The function returns an empty list when the matrix or its first row is empty, runs in O(rows * cols) time, and does not preserve row boundaries since runs may span across rows.
