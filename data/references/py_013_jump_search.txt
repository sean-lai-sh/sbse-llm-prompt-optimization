Performs jump search on a sorted list of integers for a target, returning its index or None if absent. It advances in fixed blocks of size sqrt(n) until it finds a block whose end value is >= target, then performs a linear scan inside that block to locate the target. Time complexity is O(sqrt(n)) with O(1) extra space, the input must be sorted in ascending order, and the function returns None if the jump pointer overruns the array before bracketing the target.
