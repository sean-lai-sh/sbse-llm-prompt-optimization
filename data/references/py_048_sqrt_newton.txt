Computes the square root of a non-negative number n using Newton's method (the Babylonian iteration) to within an absolute tolerance, returning the result as a float. Starting from x = n, it repeatedly updates x via x_new = (x + n / x) / 2 and stops when |x_new - x| < tolerance, returning x_new. Convergence is quadratic so it runs in O(log log(1/tolerance)) iterations with O(1) space; the function raises ValueError for negative n and short-circuits to return 0.0 for n == 0 to avoid a division-by-zero.
