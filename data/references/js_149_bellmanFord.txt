Computes single-source shortest paths from source over a directed edge list, supporting negative edge weights via the Bellman-Ford algorithm. It relaxes every edge up to n-1 times with an early-exit when no relaxation occurs in a pass, then performs an extra relaxation pass to detect negative cycles, marking any node still relaxable as belonging to a negative cycle and propagating -Infinity to all nodes reachable from such nodes via a BFS over the edge list. The function returns {dist, pred, hasNegativeCycle, negCycleEdges, negCycleNodes, getPath}, where getPath reconstructs source-to-target paths by following pred and returns null for unreachable targets, those tainted by a negative cycle, or pred chains that don't trace back to source. Worst-case complexity is O(n * |edges|) for relaxation plus O(n * |edges|) for negative-cycle propagation.
