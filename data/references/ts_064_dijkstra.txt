Computes single-source shortest paths from source on a weighted directed adjacency list graph using Dijkstra's algorithm, returning distances and a previous predecessor array suitable for path reconstruction. It uses a visited bitmap to skip nodes once finalized and a "priority queue" implemented as an array that is re-sorted on every extract-min, giving an overall O((V+E) * V log V) worst case rather than the textbook O((V+E) log V) heap variant. Unreached nodes retain Infinity in dist and null in prev; the algorithm assumes non-negative edge weights and produces incorrect results in the presence of negative edges.
