Applies a Caesar cipher to str by shifting each ASCII letter forward by shift positions modulo 26, preserving case and leaving non-letters unchanged. Normalizes shift via ((shift % 26) + 26) % 26 so negative or large shifts behave correctly, then uses a regex replacement that selects the appropriate base of 97 for lowercase or 65 for uppercase before remapping via charCodeAt and fromCharCode. Runs in O(n) time and O(n) space, returns a new string, and supports only the basic ASCII Latin alphabet.
