Performs A* shortest-path search from start to end on a 4-connected grid where cells with value 1 are obstacles, using Manhattan distance as the heuristic and uniform step cost of 1. It maintains an open list (re-sorted by f = g + h on each pop), a closed Set keyed by "row,col", and a gScore map; for each expanded node it considers the four cardinal neighbors, skips out-of-bounds, blocked, or closed cells, and pushes a new node when the tentative g improves on the prior best. On reaching end it walks parent pointers to reconstruct the path, returning {path, found: true, cost: g}; if the open set empties without success it returns an empty path with cost Infinity.
