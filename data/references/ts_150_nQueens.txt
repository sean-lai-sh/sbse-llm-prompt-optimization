Solves the N-Queens problem via depth-first backtracking, returning all valid board configurations as arrays of "Q"/"." string rows along with the solution count. It places one queen per row, using three Sets (cols, diag1 keyed by row - col, diag2 keyed by row + col) for O(1) attack checks and undoing each placement on the backward pass. Returns an empty solution list when n is 0, and runs in roughly O(n!) time in the worst case while using O(n) auxiliary space for the recursion plus the constraint sets.
