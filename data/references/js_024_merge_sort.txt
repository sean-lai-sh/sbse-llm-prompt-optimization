Recursively sorts the input array arr in ascending order using top-down merge sort that splits the array at its midpoint and merges the sorted halves with a helper that compares elements via <= to maintain stability. Runs in O(n log n) time in all cases and O(n) auxiliary space due to slice and the result arrays. Returns the sorted array without mutating the input, and short-circuits when the array length is 0 or 1 by returning it directly.
