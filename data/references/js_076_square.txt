Returns the square of the numeric input n by computing n * n directly, in O(1) time and space. The function performs no type validation or coercion, so non-numeric inputs follow JavaScript's standard arithmetic coercion (e.g. strings convert to numbers when possible, otherwise the result is NaN). It correctly handles negative numbers (always returning a non-negative result for real numbers) and floats. Very large magnitudes may overflow to Infinity per IEEE 754 double-precision limits.
