Performs A* shortest-path search on a 2D grid from start to goal where cells with value 0 are passable and movement is restricted to the four cardinal neighbors with unit cost. It maintains g_score and f_score dicts, a came_from map for path reconstruction, and a heapq-based open set keyed by f-score, defaulting the heuristic to Manhattan distance when none is supplied; on popping the goal it walks came_from backward and reverses to produce the path. The function returns None when the grid is empty or the goal is unreachable, uses a closed_set to avoid reprocessing nodes, and assumes a consistent heuristic so that closed nodes never need to be reopened.
