Performs a topological sort of a directed graph given as an adjacency list using Kahn's BFS algorithm, returning a TopoSortResult with the linear ordering and a hasCycle flag. It first computes in-degrees in O(V+E), seeds a queue with all zero-in-degree nodes, then repeatedly dequeues a node, appends it to the order, and decrements its successors' in-degrees, pushing any that reach zero. Total time is O(V+E) and space O(V); if fewer than n nodes are processed the graph contains a cycle, in which case the function returns {order: [], hasCycle: true}.
