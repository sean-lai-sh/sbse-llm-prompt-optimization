Computes the maximum value of every contiguous window of size k over a numeric array using a monotonic decreasing deque of indices. For each new index it evicts indices that have fallen out of the window from the front, pops smaller values from the back, then pushes the current index; once the first full window is reached the front index's value is appended to the result. The amortized cost is O(n) since each index is pushed and popped at most once, although the use of Array#shift makes the front eviction O(window) in the worst case in JavaScript. Empty arrays or non-positive k return [].
