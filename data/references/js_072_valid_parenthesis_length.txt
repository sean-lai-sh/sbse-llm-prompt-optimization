Returns the length of the longest valid (well-formed) parentheses substring of s using O(n) time and O(n) space dynamic programming where dp[i] holds the length of the longest valid substring ending exactly at index i. When s[i] is ')', it handles two cases: a direct "()" match adds 2 to dp[i-2], and a "...))" closing the most recent open paren extends dp[i-1] by 2 plus any valid run preceding the matched '(' at index i - dp[i-1] - 1. The running maxLen is updated whenever dp[i] grows. The function returns 0 for empty input and naturally handles strings with only '(' or unmatched closers.
