Computes the length of the longest strictly increasing subsequence of nums using the patience-sorting technique in O(n log n) time and O(n) auxiliary space. It maintains a tails array where tails[i] is the smallest possible tail of any increasing subsequence of length i+1, and for each input number performs a binary search for the leftmost position whose value is greater than or equal to it, overwriting that slot to keep tails minimal. The final answer is tails.length; the function returns 0 for an empty input and reports only the length, not the actual subsequence.
