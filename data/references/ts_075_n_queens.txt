Enumerates all valid placements of n non-attacking queens on an n x n board via row-by-row backtracking, returning an NQueensResult with totalSolutions and a list of solutions where each solution is an array of {row, col} positions. Constant-time conflict checks are achieved by maintaining three Sets tracking occupied columns, the row-col diagonals, and the row+col anti-diagonals; on each row the search tries every column, recurses on success, and undoes the placement on backtrack. When row reaches n a deep-copied snapshot of current is appended; the function throws RangeError for n <= 0 and has worst-case time roughly O(n!) due to the backtracking search.
