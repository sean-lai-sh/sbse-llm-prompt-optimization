Returns the transpose of a 2D matrix represented as a list of lists, swapping rows and columns by building a new cols-by-rows nested list comprehension. An empty matrix or a matrix whose first row is empty returns an empty list. Runs in O(rows * cols) time and allocates a new list rather than transposing in place; assumes all rows have the same length as matrix[0] and will produce a ragged or incorrect result for jagged inputs.
