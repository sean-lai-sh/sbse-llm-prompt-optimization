Sorts the integer array arr in ascending order using counting sort by first computing the value range [min, max], tallying occurrences in a count array, converting it to a prefix-sum cumulative count, then placing each element from right to left into an output array to preserve stability. Runs in O(n + k) time and O(n + k) space where k = max - min + 1, returning a new array and short-circuiting on empty input. Requires integer-valued elements and becomes memory-prohibitive when the value range vastly exceeds n.
