Implements Dijkstra's single-source shortest path algorithm on an adjacency-list graph keyed by node with [neighbor, weight] entries, returning an object containing dist and prev maps plus a getPath(target) helper that reconstructs the path by walking prev pointers. It uses a sorted-array-as-priority-queue: each iteration sorts the queue and shifts the minimum, giving an O((V+E) * V log V) worst case rather than the optimal heap-based O((V+E) log V). A visited Set guards against reprocessing nodes, and stale queue entries for already-settled nodes are skipped. Edges with negative weights would silently produce wrong results, missing graph keys are tolerated, and getPath returns an empty array when the target is unreachable.
