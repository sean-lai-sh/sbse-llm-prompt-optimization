Determines whether parentheses, square brackets, and curly braces in a string are properly matched and nested by walking the string with an explicit stack of opening characters. It returns true only when every closing bracket pops its matching opener and the stack is empty at the end, taking a string and returning a boolean in O(n) time and O(n) space. Non-bracket characters are ignored entirely, and the function correctly returns false on mismatched pairs, an unexpected closer with an empty stack, or unclosed openers remaining at the end.
