Sorts a non-negative integer array in ascending order using LSD radix sort, repeatedly applying a stable digit-wise counting sort for each base-10 place from ones up through the most significant digit of the maximum. It takes a number[] and returns a new number[] in O(d * (n + 10)) time and O(n + 10) auxiliary space, where d is the number of digits in the maximum value. Empty input is returned immediately; negative numbers and non-integers are not supported and would corrupt the digit extraction.
