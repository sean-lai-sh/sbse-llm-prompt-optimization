Converts a temperature given in degrees Celsius to degrees Fahrenheit using the standard linear formula F = celsius * 9 / 5 + 32, in O(1) time and space. It returns a floating-point number and performs no input validation, so non-numeric arguments propagate JavaScript's arithmetic coercion (yielding NaN when coercion fails). It correctly handles negative temperatures and the freezing point (0 -> 32) as well as the boiling point (100 -> 212). Operator precedence ensures the multiplication and division evaluate left-to-right before the +32 offset is applied.
