Repeats a string a given number of times by delegating to String.prototype.repeat after coercing the count to a non-negative integer with Math.max and Math.floor. It takes a string and a numeric times argument and returns a string in O(times * s.length) time and space. Negative or fractional counts are normalized to a safe non-negative integer, which avoids the RangeError that String.prototype.repeat would otherwise throw.
