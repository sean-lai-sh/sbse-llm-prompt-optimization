Performs a jump search for target in a sorted array arr by advancing in fixed blocks of size floor(sqrt(n)) until the block end exceeds target, then linearly scanning the previous block for a match. Returns the index of the first occurrence found or -1 if absent, running in O(sqrt(n)) time and O(1) space. Requires arr to be sorted in ascending order and is undefined for unsorted input; when target is greater than all elements, the linear scan still bounds itself by min(curr, n) to avoid out-of-range reads.
