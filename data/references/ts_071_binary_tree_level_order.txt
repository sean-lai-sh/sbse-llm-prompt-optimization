Performs an iterative breadth-first level-order traversal of a binary tree rooted at root and returns the values grouped per level as a number[][]. It uses an explicit queue array and processes one full level at a time by snapshotting queue.length at the start of each iteration so children pushed during that pass are deferred to the next level. Runs in O(n) time over n nodes; queue.shift is O(n) per call which makes the practical cost O(n^2) for very wide trees, and a null root yields an empty result.
