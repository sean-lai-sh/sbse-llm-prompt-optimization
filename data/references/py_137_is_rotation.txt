Determines whether string s2 is a rotation of string s1 by exploiting the property that any rotation of s1 must appear as a contiguous substring of s1 + s1. The function first rejects unequal lengths in O(1), treats two empty strings as a valid rotation, and then performs the substring check, which runs in O(n) using Python's underlying string search. Total complexity is O(n) time and O(n) extra space for the doubled string, and the comparison is case-sensitive.
