Recursively compares two values for structural equality and returns a boolean. It short-circuits on strict equality, then rejects mismatched types, null, and array-vs-object shape differences before recursing element-by-element on arrays of equal length and key-by-key on plain objects with the same number of own keys. Object key membership is checked with hasOwnProperty so inherited properties are ignored. The function does not handle cyclic references (it will recurse infinitely), nor does it special-case Date, RegExp, Map, Set, or typed arrays, which it treats as generic objects via Object.keys.
