Returns the absolute value of a number n using a ternary expression that negates the value when it is strictly less than zero and otherwise returns it unchanged, running in O(1) time and space. It correctly handles 0 (returned as +0 since 0 < 0 is false) and positive numbers, and negates negatives. For NaN it returns NaN unchanged because NaN < 0 evaluates false. It does not call Math.abs and so will not normalize -0 to +0.
