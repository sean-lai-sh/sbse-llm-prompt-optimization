Computes the connected components of an undirected graph on n nodes given a list of edges, using a union-find structure with path compression (via parent[x] = parent[parent[x]] halving) and union by rank. It unions every edge endpoint pair, then groups nodes by their final root and returns the components as a sorted list of sorted node lists. Both internal find and union run in near-amortized O(alpha(n)) time, and isolated nodes appear as singleton components in the output.
