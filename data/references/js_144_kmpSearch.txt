Finds all starting indices where pattern occurs in text using the Knuth-Morris-Pratt algorithm. It first builds the longest-proper-prefix-also-suffix (LPS) table for the pattern in O(m), then walks text and pattern with two indices, advancing both on match, falling back via lps[pi-1] on mismatch when pi > 0, and advancing only the text index when pi is already 0. Each pattern match is recorded and the search continues by setting pi = lps[pi-1] to find overlapping matches. Empty pattern, empty text, or pattern longer than text return [], yielding overall O(n + m) time and O(m) extra space.
