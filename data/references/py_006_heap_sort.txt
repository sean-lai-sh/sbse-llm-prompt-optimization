Sorts a list of integers in ascending order using an in-place heap sort, returning the mutated list. A nested recursive heapify helper enforces the max-heap property at index i over a heap of size n; the outer routine first builds a max heap by heapifying from the last non-leaf node down to the root, then repeatedly swaps the root with the last unsorted element and re-heapifies the shrunken prefix. The algorithm runs in O(n log n) time in all cases with O(1) auxiliary space (ignoring recursion stack) and is not stable.
