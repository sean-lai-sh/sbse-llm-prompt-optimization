Returns a new array sorted by a single property of each element, taking the source array, the key to compare, and an optional ascending flag (default true). It clones the input via spread to avoid mutation, then uses Array.prototype.sort with a comparator that performs < and > comparisons on the values at that key, flipping the sign when asc is false. Comparison relies on JavaScript's default relational operators, so mixed-type or non-comparable values follow standard coercion rules and equal values preserve insertion order via TimSort stability.
