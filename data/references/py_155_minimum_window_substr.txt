Finds and returns the shortest contiguous substring of s containing every character of t with required multiplicities, or None if no such window exists. It uses a sliding window with a Counter-based deficit map and a missing counter, expanding the right edge to satisfy demand and then shrinking the left edge while the window remains valid to record the smallest interval. Empty s or empty t short-circuit to None, and the algorithm runs in O(len(s) + len(t)) time using O(distinct chars) extra space.
