Converts a camelCase string str into snake_case by inserting an underscore before each uppercase ASCII letter via a regex replacement, lowercasing the entire result, and stripping any leading underscore that arose when the original string began with a capital. Returns a new string in O(n) time and O(n) space without mutating the input. Only handles A-Z as word boundaries, so digits, non-ASCII uppercase letters, and existing underscores in the input are not specially treated.
