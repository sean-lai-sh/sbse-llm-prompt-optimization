Recursively flattens an arbitrarily nested input array arr into a single-level array using an iterative stack-based traversal that pops items, spreads nested arrays back onto the stack, and unshifts non-array leaves onto the front of the result to recover the original left-to-right order. Returns a new array without mutating the input and avoids recursion-depth limits. Runs in O(n^2) time in the worst case because unshift is an O(n) operation per leaf, and uses O(n) auxiliary space for the stack and result.
