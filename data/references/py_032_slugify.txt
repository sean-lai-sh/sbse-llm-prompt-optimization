Converts an arbitrary text string into a URL-friendly slug and returns it. It lowercases and trims the input, removes any character that is not a word character, whitespace, or hyphen, then collapses any run of whitespace, underscores, or hyphens into a single hyphen and finally strips leading/trailing hyphens. The function relies on three regex passes for an O(n) transformation; because \w in Python regex includes Unicode word characters, accented letters are preserved rather than transliterated to ASCII.
