Converts a positive integer num to its Roman numeral string representation using a greedy algorithm over a fixed table of values and symbols that includes subtractive pairs (CM, CD, XC, XL, IX, IV). For each value in descending order it appends the corresponding symbol while subtracting the value from num, terminating when num reaches 0. Runs in O(1) time relative to numerals produced; it does not handle zero or negative inputs (returns an empty string for 0) and produces incorrect output for very large numbers since values cap at 1000.
