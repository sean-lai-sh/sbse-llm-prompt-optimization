Strips HTML markup from a string and decodes a small set of named entities, returning a plain-text approximation of the input. It removes script blocks, style blocks, and HTML comments via non-greedy regexes, then strips any remaining tags matching /<[^>]+>/g and decodes &lt;, &gt;, &amp;, &quot;, and &#039; before trimming. Non-string input yields the empty string. Because it is regex-based, it is not a robust XSS sanitizer: malformed tags, unencoded "<" inside attributes, or nested constructs can defeat it.
