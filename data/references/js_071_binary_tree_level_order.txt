Performs a breadth-first traversal of a binary tree rooted at root and returns an array of arrays where each inner array holds the node values at one level, top-down. It uses an explicit FIFO queue seeded with the root and, on each iteration, snapshots the queue length to consume exactly one level's worth of nodes while enqueuing their non-null left/right children. Time complexity is O(n) and space is O(w) where w is the maximum tree width; the implementation also exports the TreeNode class used to construct trees. Returns an empty array when root is null/undefined; note that queue.shift() is O(n), so very large trees incur quadratic queue cost.
