Returns a new list containing the first occurrence of each item from lst while preserving order, using a set of seen keys for O(1) membership checks. Unhashable items (lists and dicts) are tracked by their str() representation as a fallback so they can still be deduplicated, though this can incorrectly conflate distinct values that share the same string form. Runs in O(n) average time and O(n) extra space; nested structures other than list and dict (e.g. set) will raise TypeError when added to the seen set.
