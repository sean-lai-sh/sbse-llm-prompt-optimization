Computes all-pairs shortest paths in an undirected weighted graph using the Floyd-Warshall dynamic-programming algorithm, returning a num_nodes x num_nodes distance matrix or None if a negative cycle is detected. It initializes the matrix with infinities, zeros on the diagonal, and the minimum of the original and given weight for each edge in both directions, then performs the classic triple-nested loop relaxation dist[i][j] = min(dist[i][j], dist[i][k] + dist[k][j]). Runtime is O(V^3) and space is O(V^2); a negative value on any diagonal entry after relaxation triggers the None return.
