Recursively determines structural equality between two values of the same generic type T. After a strict equality short-circuit and null/typeof guards, it compares own enumerable keys via Object.keys, returning false on length mismatch or any missing key, and otherwise recurses on each property. The implementation does not handle cyclic references (will infinitely recurse), treats arrays and plain objects identically by their string keys, and does not special-case Date, Map, Set, or RegExp instances.
