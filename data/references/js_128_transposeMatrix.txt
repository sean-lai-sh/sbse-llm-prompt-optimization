Returns the transpose of a 2D matrix as a freshly allocated cols-by-rows array, leaving the input untouched. It infers row and column counts from matrix.length and matrix[0].length, allocates a zero-filled output via Array.from, and assigns result[j][i] = matrix[i][j] in a double loop. Time and space complexity are both O(rows * cols). Non-array or empty inputs return []; the function assumes a rectangular matrix and will produce truncated or undefined entries if rows have inconsistent lengths.
