Generates the n-th row of Pascal's triangle (zero-indexed) as a list of n+1 binomial coefficients C(n, 0) through C(n, n). It computes each entry iteratively from the previous one using the multiplicative recurrence C(n, k) = C(n, k-1) * (n - k + 1) / k with integer division, achieving O(n) time and O(n) space. The function returns [1] for n = 0 and assumes a non-negative integer input; negative n yields just [1] because the loop body does not execute.
