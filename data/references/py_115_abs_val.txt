Returns the absolute value of numeric input n by returning n unchanged when it is non-negative and -n otherwise, using a conditional expression rather than the built-in abs. Runs in O(1) time and handles zero by returning it unchanged via the n >= 0 branch. For floats it preserves the IEEE 754 sign of -0.0 by negating it to +0.0, but it does not handle NaN specially since NaN >= 0 is False so the function returns -NaN.
