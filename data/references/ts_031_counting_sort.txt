Sorts a numeric array in ascending order with counting sort by tallying occurrences of each integer in the inclusive range [min, max] and emitting them in order. It takes a number[] and returns a new number[] in O(n + k) time and O(k) space, where k = max - min + 1, returning an empty array immediately for empty input. It supports negative integers via the min offset but assumes integer values; non-integer inputs would yield incorrect indices and skewed results.
