Tokenizes a text string and returns a dict mapping each cleaned word to the number of times it occurs. It lowercases the text, splits on whitespace, strips out non-alphanumeric characters from each token, and increments the count only for non-empty cleaned words using dict.get with a default of 0. Time and space are O(n) in the input length, the comparison is case-insensitive, punctuation is removed (so "Hello!" and "hello" count as the same word), and tokens that consist entirely of punctuation are ignored.
