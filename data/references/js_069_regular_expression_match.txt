Implements regex matching with support for '.' (any single character) and '*' (zero or more of the preceding element) using bottom-up dynamic programming, where dp[i][j] is true when s[0..i-1] fully matches p[0..j-1]. The base row is initialized so patterns of the form a*b*c* can match the empty string by propagating dp[0][j-2] when p[j-1] is '*', and each cell handles '*' as either zero occurrences (dp[i][j-2]) or one-or-more (dp[i-1][j] when the preceding pattern char matches s[i-1] or is '.'); literal/dot matches consume one character via dp[i-1][j-1]. Time and space are O(m*n). It assumes patterns are well-formed (every '*' has a preceding element) and matches the entire string rather than a substring.
