Performs an iterative binary search over a sorted numeric array, narrowing the half-open range with low and high pointers and using an unsigned right shift to compute the midpoint without overflow. It accepts a number[] and a target number and returns the matched index or -1 if the value is absent, running in O(log n) time and O(1) space. The function assumes the input is sorted in ascending order; behavior is undefined on unsorted data, and an empty array immediately returns -1.
