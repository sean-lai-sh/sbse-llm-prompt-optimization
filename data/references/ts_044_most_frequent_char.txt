Returns the most frequently occurring character in a string by tallying counts in a Map and then scanning the Map to find the entry with the largest count. It takes a string and returns a single-character string in O(n) time and O(k) space, where k is the size of the character alphabet, and throws a RangeError for empty input. Ties are broken by the iteration order of the Map, which mirrors first-occurrence insertion order, so the earliest-appearing character among the maxima wins.
