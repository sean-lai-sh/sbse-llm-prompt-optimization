Returns the nth Fibonacci number for non-negative integer n using an iterative two-variable approach. For n <= 1 it returns n directly; otherwise it loops from index 2 to n, advancing the (a, b) pair via tuple assignment a, b = b, a + b, and finally returns b. Time complexity is O(n) with O(1) extra space, the function raises ValueError for negative n, and it leverages Python's arbitrary-precision integers so it never overflows.
