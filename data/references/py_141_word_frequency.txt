Computes a case-insensitive word frequency map for string s by lowercasing the input, splitting on whitespace, and counting occurrences of each token in a dict using dict.get with a default of 0. Returns a Dict[str, int] mapping each lowercase word to its count, runs in O(n) time and O(k) space where n is character count and k is the number of distinct words. The function does not strip punctuation, so tokens like "hello," and "hello" are counted separately, and an empty or whitespace-only input yields an empty dict.
