Returns the median of an already-sorted numeric array sorted in O(1) time by indexing the middle element directly. For odd-length arrays it returns sorted[mid] where mid = floor(len/2); for even-length arrays it averages the two central elements sorted[mid-1] and sorted[mid]. The function returns 0 for an empty array as a sentinel and does not verify that the input is actually sorted, so unsorted input silently produces a wrong answer.
