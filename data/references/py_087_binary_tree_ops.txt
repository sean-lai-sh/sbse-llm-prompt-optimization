Dispatches one of five binary-search-tree operations (insert, search, inorder, height, delete) against a tree represented as nested dicts with 'value', 'left', and 'right' keys, returning the updated tree, a bool, a sorted list, an int height, or the modified tree respectively. Insert and delete return the (possibly new) root and follow standard BST recursion; deletion of a two-child node uses the in-order successor (leftmost of the right subtree) to replace the value and then deletes the successor. Each recursive operation runs in O(h) for balanced trees but O(n) worst case on skewed trees, raises ValueError for unknown operation names, and equal values are treated as already-present (no duplicates inserted).
