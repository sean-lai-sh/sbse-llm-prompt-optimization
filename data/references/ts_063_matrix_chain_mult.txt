Solves the matrix-chain multiplication optimization problem given a dims array where matrix i has shape dims[i] x dims[i+1], filling an n by n cost table m and split table s by increasing chain length in O(n^3) time and O(n^2) space. For each subchain (i,j) it tries every split point k and records the minimum scalar-multiplication cost m[i][k] + m[k+1][j] + dims[i]*dims[k+1]*dims[j+1] along with the optimal k in s[i][j]. Returns a MatrixChainResult with minOperations and a recursively constructed parenthesization string of the form (M1 x (M2 x M3)); throws RangeError when fewer than one matrix is provided (n <= 0).
