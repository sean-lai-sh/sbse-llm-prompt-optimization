Generates an array of the first n Fibonacci numbers iteratively, starting from the seed [0, 1] and appending each subsequent term as the sum of the two preceding values. Runs in O(n) time and O(n) space for the returned array. Edge cases are handled explicitly: n <= 0 returns an empty array, and n === 1 returns [0]. For very large n, intermediate values exceed Number.MAX_SAFE_INTEGER and lose precision because the function uses standard JavaScript numbers rather than BigInt.
