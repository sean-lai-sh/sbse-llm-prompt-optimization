Implements Knuth-Morris-Pratt substring search, returning every starting index where pattern occurs in text along with the precomputed LPS (longest proper prefix that is also a suffix) array. It first builds the LPS array in O(m) by tracking the current matched-prefix length and falling back via lps[len - 1] on mismatches, then scans the text in O(n) using the LPS to skip rather than restart on a mismatch and recording matches at i - j when j reaches the pattern length. Returns empty matches and an empty LPS for an empty pattern, and runs in overall O(n + m) time and O(m) auxiliary space.
