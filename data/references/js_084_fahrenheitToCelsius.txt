Converts a temperature given in degrees Fahrenheit to degrees Celsius using the inverse linear formula C = (fahrenheit - 32) * 5 / 9, in O(1) time and space. The result is a floating-point number; the function does no input validation, so non-numeric or NaN inputs propagate to a NaN result, and Infinity inputs yield +/- Infinity. It correctly returns 0 for the freezing point (32) and 100 for the boiling point (212). Subtle floating-point representation may produce results like 36.99999... rather than exact decimals.
