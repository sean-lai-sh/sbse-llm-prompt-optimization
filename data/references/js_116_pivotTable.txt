Builds a pivot-table representation of an array of records, grouping by row and column key fields and aggregating a value field. It collects unique row and column values via Set, buckets each record's value under a "row__col" composite key, then emits one row object per unique row value with one property per unique column value holding the aggregate. Supported aggregators are "sum", "avg", "count", "min", and "max", with sum as the default fallback for empty buckets returning 0. The bucket key uses literal "__" concatenation, which can collide when row or column values themselves contain "__".
