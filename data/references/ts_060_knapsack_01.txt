Solves the 0/1 knapsack problem with bottom-up DP over an (n+1) by (capacity+1) table dp where dp[i][w] is the best value achievable using the first i items within capacity w. It runs in O(n*capacity) time and space, then back-tracks from dp[n][capacity] by comparing each row with the row above it to recover which item indices were chosen, reversing the list so indices are in ascending order. Returns a KnapsackResult with maxValue and selectedItems; weights and capacity are assumed to be non-negative integers and items with weight greater than the remaining capacity are skipped.
