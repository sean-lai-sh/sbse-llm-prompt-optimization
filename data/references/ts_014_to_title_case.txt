Converts a string to title case by lowercasing it, splitting on single spaces, uppercasing the first character of each non-empty token, and joining the tokens back with single spaces. It takes a string and returns a string in O(n) time and O(n) space. Because it splits only on the literal space character, runs of multiple spaces, tabs, and other whitespace are not collapsed, and empty tokens are passed through unchanged to preserve original spacing.
