Safely parses a JSON document from a str or bytes input, returning the parsed Python value on success or a configurable default on any failure. In strict mode it raises ValueError/TypeError for None inputs, decoding errors, non-str/bytes inputs, empty strings, or invalid JSON; in non-strict mode all of these conditions silently return default. Bytes inputs are decoded with the specified encoding (default UTF-8), the input is stripped of surrounding whitespace before parsing, and json.loads handles the actual deserialization.
