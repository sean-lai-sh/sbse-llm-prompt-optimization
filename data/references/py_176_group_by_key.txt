Groups a list of dictionary records into a dict mapping each distinct value of the named key to the list of records containing that value. It iterates the records once, fetching record.get(key) (which yields None for records missing the key, lumping them into a single None bucket) and appending the record into the corresponding list. The function runs in O(n) time, preserves the input order of records within each group, and requires the key's values to be hashable.
