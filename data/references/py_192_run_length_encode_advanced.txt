Encodes an integer list using a hybrid scheme that emits [run_length, value] sublists for runs of three or more consecutive equal values and bare integers for isolated literals between runs. It scans the list once, first measuring the current run length and emitting a run-pair when it is at least 3, otherwise extending a literal block until it is followed by another run of three equal values or the input ends. The function returns an empty list for empty input, runs in O(n) time, and chooses to inline literal blocks element-by-element rather than wrapping them.
