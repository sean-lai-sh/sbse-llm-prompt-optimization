Parses a URL query string into a Record<string, string>, optionally stripping a leading "?" and returning an empty object for empty input. It splits on "&", locates the first "=" in each pair, and decodeURIComponent-decodes the key and value separately; pairs without "=" map the decoded key to an empty string. Later occurrences of a duplicate key overwrite earlier ones, and array-style keys (e.g. "a[]=1&a[]=2") are not collected into arrays.
