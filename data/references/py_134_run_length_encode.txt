Performs run-length encoding on string s by walking it once and grouping consecutive identical characters, returning a list of (character, count) tuples. Maintains a current character and counter, flushing a tuple to the result whenever the next character differs, and appends the final run after the loop ends. Runs in O(n) time and O(n) worst-case space, and short-circuits to an empty list when s is empty to safely handle the missing initial character.
