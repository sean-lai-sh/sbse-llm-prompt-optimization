Counts the occurrences of each distinct value of a chosen key across an array of objects, returning a Record<string, number> tally. It iterates the array once, coerces each item's key value to a string via String() (so numbers, booleans, null, and undefined all become string keys), and increments the running count using nullish coalescing to default missing entries to zero. Runs in O(n) time and O(g) extra space where g is the number of distinct group values.
