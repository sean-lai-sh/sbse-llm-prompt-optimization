Serializes an array of records to a CSV string by inferring column headers from the first row's keys and emitting one comma-separated line per row plus a header line. Each cell is coerced to a string (treating null/undefined as ""), and values containing commas, double quotes, or newlines are wrapped in double quotes with internal quotes doubled per RFC 4180-style escaping. Returns an empty string for an empty input array, and silently drops fields present in later rows but absent from the first row's key set.
