Implements regular-expression matching of string s against pattern p where '.' matches any single character and '*' matches zero or more of the preceding element, using bottom-up DP over an (m+1) by (n+1) boolean table. The first row is seeded so that patterns like a*b*c* can match the empty string, and each cell handles '*' by checking either the zero-occurrence case dp[i][j-2] or the one-or-more case dp[i-1][j] when the preceding pattern character is '.' or matches s[i-1]. Runs in O(m*n) time and space and returns dp[m][n]; assumes '*' is always preceded by a valid character and does not validate the pattern.
