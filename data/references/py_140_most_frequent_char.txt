Returns the character that appears most frequently in string s by building a frequency dictionary in a single pass and then taking max over its keys with a value-based key function. Runs in O(n) time and O(k) space where n is len(s) and k is the number of distinct characters, and returns the empty string for empty input. On ties, max returns the first key with the highest count according to dict insertion order, which corresponds to the first character in s to reach that maximum.
