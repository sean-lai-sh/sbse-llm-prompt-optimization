Builds a Huffman code for the input text and returns the per-character code Map, the encoded bit string, and the constructed binary tree. It first tallies character frequencies, creates a leaf node per character, sorts the node list ascending by frequency, then repeatedly removes the two lowest-frequency nodes, merges them under a new internal node whose frequency is their sum, and inserts the merge back into the (still) sorted list via linear scan; codes are then assigned via DFS where left edges append "0" and right edges append "1". Returns empty results for empty text and special-cases a single distinct character by emitting code "0", with overall complexity dominated by the O(k^2) sorted-list maintenance for k unique characters.
