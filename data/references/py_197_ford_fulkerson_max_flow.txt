Computes the maximum flow from source to sink in a directed graph (the Edmonds-Karp variant of Ford-Fulkerson) given a node count and a list of (u, v, capacity) edges. It builds an undirected adjacency list and a (u, v) -> capacity dict (summing parallel edges), then repeatedly finds a shortest augmenting path via BFS, sends the bottleneck capacity along it, and updates forward and reverse residual capacities until no augmenting path exists. The algorithm runs in O(V * E^2) time, returns 0 when source equals sink or sink is unreachable, and assumes capacities are non-negative.
