Recursively flattens a nested dictionary into a single-level dict whose keys are dot-joined paths from the root to each leaf value. It walks each item, building full_key as prefix.key when a prefix is supplied, and recurses into nested dicts while copying any non-dict value as a leaf. The function does not descend into lists or other containers and would silently overwrite entries when a leaf path collides with the dotted name of a sibling subtree.
