Determines whether the string s can be segmented into a sequence of words from word_dict, returning a bool when return_sentences is False or a list of all valid space-separated segmentations when True. The boolean mode uses a 1D DP array where dp[i] is True if some prefix break point j gives dp[j] and s[j:i] in the word set, running in O(n^2) time after building a set from the dictionary. The sentences mode uses recursive backtracking that can run in exponential time; both modes return False or [] respectively for an empty input string.
