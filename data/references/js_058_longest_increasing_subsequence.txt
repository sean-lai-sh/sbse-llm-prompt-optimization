Computes a longest strictly increasing subsequence of nums using the classic O(n^2) dynamic programming formulation, where dp[i] holds the LIS length ending at index i and parent[i] records the predecessor index for reconstruction. It scans every prior index j for each i, extending dp[i] when nums[j] < nums[i] improves the length, and tracks the global maxLen and maxIdx. After filling the tables it walks parent pointers from maxIdx back to -1, unshifting values to rebuild the subsequence in order. It returns an empty array for null or empty input and, in case of ties, returns the subsequence ending at the earliest maximum index encountered.
