Performs a one-level structural comparison of two values of the same generic type T using strict equality on each own enumerable property. It first short-circuits on a === b, then rejects non-object operands and nulls, then verifies that both objects have the same number of keys and that every key in a maps to a value identical to b's value via ===. Runs in O(n) over the key count and does not recurse into nested objects, so equal-but-distinct nested references are reported as unequal.
