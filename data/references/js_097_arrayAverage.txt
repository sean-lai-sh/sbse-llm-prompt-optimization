Computes the arithmetic mean of the numeric array arr by reducing it to a sum (initial value 0) and dividing by arr.length, in O(n) time and O(1) extra space. The empty-array case is short-circuited to return 0, avoiding the 0 / 0 -> NaN result. Non-numeric elements are coerced via the + operator, which can introduce NaN or string concatenation surprises if elements are not all numbers. Throws a TypeError if arr is null or undefined since it accesses .length and .reduce directly.
