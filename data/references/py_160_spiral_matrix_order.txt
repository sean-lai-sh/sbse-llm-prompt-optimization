Traverses a 2D matrix in clockwise spiral order starting from the top-left corner and returns the visited values in a flat list. It maintains four shrinking boundaries (top, bottom, left, right) and walks each layer in four directions, guarding the bottom and left passes with bounds checks to avoid double-visiting rows or columns when a single row or column remains. The algorithm handles empty matrices and non-square shapes, runs in O(rows * cols) time, and allocates O(rows * cols) space for the output.
