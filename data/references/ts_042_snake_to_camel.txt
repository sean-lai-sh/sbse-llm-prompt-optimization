Converts a snake_case identifier to camelCase by first lowercasing the entire string and then replacing each underscore-followed-by-lowercase-letter pattern with the uppercase letter alone. It takes a string and returns a string in O(n) time and O(n) space. Trailing underscores or underscores not followed by a-z are preserved verbatim, and the leading character is left lowercase so the result is camelCase rather than PascalCase.
