Merges any overlapping intervals in the input array and returns a new array of disjoint intervals sorted by start. It first creates a shallow copy of the input and sorts that copy by start in O(n log n) time, then performs a single pass that either extends the last merged interval's end via Math.max or appends a fresh shallow clone of the current interval. The original input is not mutated, intervals that merely touch (current.start === last.end) are treated as overlapping, and an empty input returns an empty array.
