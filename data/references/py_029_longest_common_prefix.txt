Returns the longest common prefix shared by all strings in the input list, or '' if the list is empty or no characters are shared. Starting with the first string as a candidate prefix, it iterates through the remaining strings and trims one character off the right of the prefix until the current string starts with it, exiting early with '' if the prefix becomes empty. Worst-case time is O(N * M) where N is the number of strings and M is the length of the shortest, and space is O(M); empty input and empty strings within the list are handled correctly via the startswith check and the truncation loop.
