Extracts all integer and decimal numeric literals from string s using the regular expression r'-?\d+(?:\.\d+)?', then converts each match to a float and returns the resulting list. The regex matches an optional leading minus sign, one or more digits, and an optional fractional part, so it captures negatives but not scientific notation, signed exponents, or thousands separators. Runs in roughly O(n) time over the input length and returns an empty list when no numeric tokens are present.
