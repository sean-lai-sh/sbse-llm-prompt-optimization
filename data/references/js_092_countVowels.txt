Counts the number of ASCII vowels (a, e, i, o, u) in str using a case-insensitive global regex match and returning the resulting array's length, in O(n) time and O(k) extra space for the matched array where k is the vowel count. The fallback `|| []` ensures a 0 result when str.match returns null (no matches). It only counts the five English vowels and does not consider 'y' or accented vowels (e.g. 'a', 'e', 'u'), nor uppercase Cyrillic or other scripts. The function will throw a TypeError if str is null or undefined since it calls .match on it directly.
