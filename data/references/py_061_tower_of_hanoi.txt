Recursively solves the Tower of Hanoi puzzle for n disks, returning the ordered list of (source, target) peg-name tuples representing each move. The base case n == 1 produces a single direct move, and the recursive case moves n-1 disks to the auxiliary peg, moves the largest disk to the target, and then moves the n-1 disks from auxiliary to target. The algorithm produces 2^n - 1 moves and uses O(n) recursion depth; it does not validate that n is positive and will return an empty list for n == 0 only via the recursive concatenation pattern (actually it will recurse indefinitely for n <= 0).
