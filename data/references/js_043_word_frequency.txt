Builds a frequency map of words appearing in the input string str by lowercasing it, extracting word tokens with the regex /\b\w+\b/g, and tallying occurrences in a plain object using a for-of loop. Returns the map keyed by word with integer counts, defaulting to an empty object when no word characters are present. Runs in O(n) time and O(k) space where k is the number of distinct words; relies on \w which only covers ASCII letters, digits, and underscore, so non-ASCII words are not tokenized correctly.
