Returns a new list where each element is the product of all other elements of the input list, computed in O(n) time without using division. It performs two linear passes: a forward pass that fills each slot with the prefix product to its left, and a backward pass that multiplies in the suffix product to its right, using only O(1) auxiliary scalars beyond the output. For an empty input it returns an empty list, and lists containing zeros are handled correctly because no division is involved.
