Performs a safe dotted-path lookup on a nested object, taking a record and a dot-delimited path string and returning the value cast to T or undefined if any segment is missing. It splits the path on "." and iteratively walks the keys, short-circuiting to undefined whenever the current node is null, undefined, or a non-object. Runs in O(d) time where d is path depth, but does not support array indices, bracket notation, or escaped dots in keys.
