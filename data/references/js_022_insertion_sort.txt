Sorts a shallow copy of arr in ascending order with the classic insertion sort, scanning from the second element and shifting larger predecessors to the right until the correct insertion point is found for each key. Returns the sorted copy without mutating the original input array. Runs in O(n^2) worst- and average-case time, O(n) on nearly sorted input, with O(n) auxiliary space due to the copy and using the > operator for comparisons, which makes it stable.
