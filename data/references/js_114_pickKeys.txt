Builds a new object containing only the requested own properties of the input object. It accepts obj and a single key or array of keys, normalizes the keys into a Set, and copies each key whose presence is confirmed by hasOwnProperty into a fresh result object. Non-object or null inputs return {}. The implementation iterates the requested keys rather than the source object, so its complexity is O(k) in the number of requested keys and missing keys are silently skipped without producing undefined entries.
