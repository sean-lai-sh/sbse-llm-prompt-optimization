Performs a topological sort of a directed graph (adjacency map of node to neighbors) using recursive DFS with post-order insertion at the front of a stack via unshift, yielding nodes in dependency order. Cycle detection is performed with an inStack set tracked along the active recursion path, throwing an Error naming the offending node if a back edge is encountered. After the main DFS over all keyed nodes (which handles disconnected components), it appends any neighbor-only nodes that never appeared as keys to the end of the result. Time complexity is O(V+E); deep graphs may exceed the JavaScript call stack because recursion is not converted to iteration.
