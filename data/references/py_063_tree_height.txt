Recursively computes the height of a binary tree represented as nested dicts with 'left' and 'right' keys, returning 0 for a None node and 1 plus the max of the two subtree heights otherwise. It accepts an Optional[dict] root and returns an int counting nodes along the longest root-to-leaf path. Time complexity is O(n) where n is the number of nodes and recursion depth is O(h), so deeply unbalanced trees may cause stack overflow.
