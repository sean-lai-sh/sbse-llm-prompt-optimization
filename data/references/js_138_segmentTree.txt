Builds a 1-indexed segment tree over nums supporting range-sum queries and point updates, then dispatches a list of operations and returns their results. The tree array is sized 4*n and built recursively in O(n); each update or query then runs in O(log n) by recursing into the relevant child, with query returning 0 for fully-disjoint segments and the cached node sum for fully-covered segments. The dispatcher recognizes "query" (range sum) and "update" (set nums[i] = val and refresh ancestors); a rangeMin helper is defined but unreachable from the dispatcher. The function mutates the supplied nums array on updates, and assumes 0 <= l <= r < n and 0 <= i < n.
