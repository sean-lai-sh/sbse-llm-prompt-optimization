Removes all whitespace characters from str by using String.prototype.replace with the global regex /\s+/g, which matches runs of one or more whitespace (spaces, tabs, newlines, form feeds, and Unicode whitespace) and replaces them with the empty string. Runs in O(n) time and O(n) space for the new string; the original string is not mutated since strings are immutable in JavaScript. It strips not just literal spaces but also tabs and newlines, which may surprise callers expecting only ASCII space removal. Throws a TypeError if str is null or undefined.
