Computes the transpose of an arbitrary rows-by-cols numeric matrix, returning a new cols-by-rows matrix without mutating the input. It allocates the result via Array.from with cols outer entries each filled to length rows with zeros, then copies matrix[r][c] into result[c][r] using nested loops in O(rows * cols) time and matching extra space. Returns an empty array if the input has zero rows, and assumes a rectangular grid (the column count is read from row 0).
