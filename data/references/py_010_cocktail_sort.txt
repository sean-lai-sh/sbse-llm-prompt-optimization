Sorts a list of integers in ascending order using cocktail (bidirectional bubble) sort and returns the mutated input. Each iteration first sweeps left-to-right swapping adjacent out-of-order pairs to push the largest unsorted element to the end, then sweeps right-to-left to push the smallest to the start, shrinking the active window with start and end pointers and exiting early via a swapped flag if a forward pass made no swaps. Worst- and average-case time is O(n^2) with O(1) extra space, the sort is stable, and best-case is O(n) on already-sorted input thanks to the early termination.
