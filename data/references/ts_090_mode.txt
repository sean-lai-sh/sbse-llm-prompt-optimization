Returns the most frequently occurring value in the numeric array nums by building a frequency Map in one pass and then scanning the map entries to find the maximum count, running in O(n) time and O(k) space for k distinct values. Ties are broken by Map iteration order (insertion order), so the first value to reach the highest count wins. The function does not handle empty input cleanly: it initializes result to nums[0], which is undefined when nums is empty, and that undefined leaks through as the return value.
