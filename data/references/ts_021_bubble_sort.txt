Sorts a numeric array in ascending order using bubble sort with a swapped flag that allows early termination once a pass makes no swaps. It accepts a number[] and returns a new sorted number[] in O(n^2) worst-case time, O(n) best-case time on already-sorted input, and O(n) space because the array is shallow-copied first. The original input is not mutated, and the algorithm is stable since equal elements never swap.
