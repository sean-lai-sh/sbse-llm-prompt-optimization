Returns the median of a presorted numeric array sortedNums in O(1) time and space, assuming the caller has already sorted the input. For odd-length arrays it returns the middle element at index Math.floor(len/2); for even-length arrays it returns the average of the two central elements at indices mid-1 and mid. It returns undefined for empty arrays as a sentinel for "no median". Because it does not sort internally, passing an unsorted array yields an incorrect result silently.
