Performs breadth-first search from a start vertex over a Map-based adjacency-list graph and returns the visit order, the per-node hop distance, and the BFS-tree parent of each visited node (with the start mapped to null). It uses an array as a FIFO queue plus a Set of seen nodes to avoid revisits, dequeuing via Array.prototype.shift and enqueuing each unseen neighbor with distance set to the current node's distance plus one. Runs in O(V + E) logically though shift makes it O(V^2) in the worst case, and only the connected component reachable from start is explored.
